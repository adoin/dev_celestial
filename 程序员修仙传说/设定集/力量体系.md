# 力量体系设定

## 体系公理
- 核心信条/定律：
  - 空间是3维的，在每个3维坐标点上牵引不同能力
  - 境界差异=空间掌握能力（速度+容量+持续时间）
  - 例：筑基初期（10m³/s × 3.7s）→ 筑基中期（100m³/s × 40s）
- 代价规则：
  - 越级战斗消耗巨大，可能损伤根基
  - 斩灵境界：斩去人性接近神道，但大部分止步化神无法飞升
  - 智能体消耗灵石和数据，秘密暴露会被全界追杀
- 公平性原则（谁能破格、谁必须付出代价）：
  - 灵根决定修炼速度，杂灵根几乎无法突破
  - 用智能体绕过灵根限制是唯一已知的"破格"路径，但代价巨大

## 体系类型
- 体系类型：境界修炼(空间掌握能力)
- 典型境界链：练气 → 筑基 → 金丹 → 元婴 → 斩灵（可选） → 化神 → 飞升
- 小境界划分：初期/中期/后期/圆满
- **斩灵特殊性**：斩去人性接近神道的捷径（非必经），是主流但有残缺，大部分斩灵修士止步化神无法飞升

- 反套路规则如何体现：
