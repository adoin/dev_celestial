# 第46章 质疑邪术

"叮！检测到宗门长老会紧急召集令。"

"所有筑基期以上弟子，一个时辰内到议事大殿集合！"

Potato的提示音在尤溪脑海中响起。

尤溪眉头一皱。

"长老会紧急召集？发生什么事了？"

"根据宗门情报网络分析，本次召集与你的修为异常有关。"

"建议：做好应对准备。"

尤溪心中一凛。

他的修为确实涨得太快了...

从炼气三层到筑基初期巅峰，只用了短短几个月。

这在修仙界，简直是匪夷所思的速度！

"看来有人坐不住了..."

他整理了一下衣衫，向议事大殿走去。

...

议事大殿内，人头攒动。

碧寒宫的所有长老和筑基期弟子都到齐了，气氛凝重。

尤溪一走进来，就感觉到无数道目光落在自己身上。

有好奇的、有怀疑的、有幸灾乐祸的...

还有温如玉那道阴冷如毒蛇的目光。

"尤溪，到前面来。"

大长老玄清子的声音响起，语气严肃。

尤溪走上前去，恭敬行礼。

"弟子在。"

玄清子看着他，眼中带着复杂的神色。

"尤溪，本座问你——"

"你入宗时，修为是炼气三层，对吗？"

"是。"

"而现在，你的修为是？"

"筑基初期巅峰。"

"哗——"

全场哗然！

虽然很多人都知道尤溪修为提升很快，但听到他亲口承认，还是震惊不已！

短短几个月，从炼气三层到筑基初期巅峰？！

这他妈还是人吗？！

...

"安静！"

玄清子一声冷喝，全场安静下来。

他看着尤溪，继续问道：

"你的灵根，是五行杂灵根，没错吧？"

"没错。"

"五行杂灵根，是修仙界公认的最废灵根，修炼速度应该比单灵根慢五倍以上。"

"而你，不仅不慢，反而比天灵根还快..."

"你能解释一下，这是为什么吗？"

尤溪心中冷笑。

来了。

这是要兴师问罪了！

他正要开口，一个声音抢先响起——

"大长老，弟子有话要说！"

温如玉站了出来，一脸正气。

"大师兄请讲。"

"弟子认为，尤溪师弟修为暴涨，必有蹊跷！"

温如玉朗声说道，目光扫视全场。

"诸位师兄弟都知道，五行杂灵根的修炼速度极慢，这是修仙界的常识！"

"而尤溪师弟，区区几个月就从炼气三层突破到筑基初期巅峰..."

"这根本不是正常修炼能达到的速度！"

他转过身，指着尤溪，声色俱厉：

"弟子怀疑，尤溪修炼了邪术！"

"或者是...使用了禁药！"

"否则，根本无法解释他的修为异常！"

...

全场再次哗然！

邪术？！

禁药？！

这可是修仙界的大忌！

一旦坐实，轻则废除修为，重则当场击毙！

...

尤溪眯起眼睛，看着温如玉那张正义凛然的脸，心中杀意涌动。

这个狗东西...

借刀杀人玩得真溜啊！

他明明派了二十个人去追杀自己，现在反倒装起好人来了？

"温师兄此言差矣。"

尤溪淡淡开口。

"哦？"

温如玉嘴角勾起一抹冷笑。

"那尤师弟倒是说说，你的修为为何涨得这么快？"

"莫非是得了什么逆天奇遇？"

"如果是奇遇，不妨说出来让大家听听。"

"反正都是同门师兄弟，有好东西应该分享嘛。"

他这话看似大度，实则阴险至极。

如果尤溪说出寂云谷的事，肯定会暴露智能体的存在。

如果不说...

那就是默认了修炼邪术！

...

尤溪心中冷笑。

想套路我？

老子写代码的时候，你还在玩泥巴呢！

"温师兄说得对，我确实得了奇遇。"

尤溪坦然说道。

"在寂云谷任务中，我误入了一处上古遗迹，得到了五行道人的传承。"

"五行道人？！"

玄清子脸色一变。

"可是那个传说中的化神期大能？"

"正是。"

尤溪点头。

"我在遗迹中获得了《五行真经》的传承，这部功法专门适配五行灵根，修炼速度远胜普通功法。"

"所以我的修为才能提升得这么快。"

...

全场再次震惊！

五行道人的传承？！

那可是传说中的至宝啊！

难怪尤溪修为暴涨...

原来是得了这等逆天机缘！

...

温如玉脸色阴沉。

该死！

这小子竟然真的得到了奇遇！

但他不甘心！

"大长老，弟子认为此事有蹊跷！"

温如玉继续说道。

"就算尤师弟真的得了五行道人的传承，也不可能几个月就从炼气三层突破到筑基初期巅峰！"

"除非...他使用了某种邪门手段，强行提升修为！"

"否则，根本说不通！"

...

其他长老也纷纷点头。

确实，就算有顶级功法，这个速度也太夸张了！

玄清子沉吟片刻，看向尤溪：

"尤溪，你可有什么证据证明自己的清白？"

尤溪心中暗叹。

证据？

他能拿出什么证据？

总不能说"老子身体里有个智能体，可以自动优化修炼"吧？

那不被当成疯子才怪！

"大长老，弟子愿意接受调查。"

尤溪沉声说道。

"但弟子可以对天发誓，从未修炼过任何邪术，也未使用过任何禁药！"

"弟子的修为，完全是靠正常修炼得来的！"

...

温如玉冷笑。

"发誓？誓言能当证据吗？"

"尤师弟，如果你真的清白，不如让长老们检查一下你的身体。"

"看看是否有邪术或禁药的痕迹，如何？"

他这话一出，尤溪心中一凛。

检查身体？

那Potato不就暴露了吗？！

虽然Potato藏在识海深处，一般的检查发现不了...

但万一有精通神魂的长老呢？

万一被发现...

他就是有一百张嘴也说不清了！

...

就在尤溪犹豫之际，一个清冷的声音响起——

"够了。"

石鸾从人群中走出，站到尤溪身边。

"温如玉，你口口声声说尤溪修炼邪术，有证据吗？"

"没有证据，就是污蔑！"

温如玉脸色一沉。

"石师妹，我这是为了宗门的安全着想..."

"安全？"

石鸾冷笑。

"你派二十个人追杀尤溪的时候，怎么不考虑宗门的安全？"

"现在倒装起好人来了？"

...

全场再次哗然！

温如玉派人追杀尤溪？！

这可是宗门大忌啊！

...

温如玉脸色铁青。

"石师妹，你这话是什么意思？"

"我什么时候派人追杀尤师弟了？"

"你可不要乱说！"

石鸾正要反驳，玄清子摆了摆手。

"好了，都别吵了。"

他看着尤溪，沉声道：

"尤溪，本座给你三天时间。"

"三天内，你必须证明自己修为的合法性。"

"否则..."

"宗门将依法对你进行处置！"

...

尤溪心中一沉。

三天？

这个时间太紧了！

但表面上，他还是恭敬地点头。

"弟子明白。"

...

会议结束，人群散去。

尤溪走出议事大殿，石鸾追了上来。

"尤溪，你没事吧？"

她担忧地看着他。

"没事。"

尤溪摇摇头，眼中闪过一丝冷意。

"温如玉这是想借刀杀人..."

"三天时间，太短了。"

石鸾皱眉。

"你有什么办法吗？"

尤溪沉默片刻，缓缓说道：

"只有一个办法..."

"在三天内，突破到筑基中期！"

"只要我能以筑基中期的修为，正面击败温如玉..."

"就能证明我的实力是真实的，不是邪术！"

...

石鸾瞪大了眼睛。

"三天突破筑基中期？"

"这怎么可能？！"

尤溪看着她，眼中闪过一抹坚定。

"对别人来说不可能..."

"但对我来说，未必！"

"Potato，启动紧急突破计划！"

"收到！"

"紧急突破计划启动中..."

"预计突破时间：72小时！"

...

夜幕降临。

尤溪回到住处，开始最后的冲刺。

三天...

只有三天时间！

他必须突破！

否则，不仅智能体会暴露，他还会被废除修为，甚至性命不保！

温如玉...

你给我等着！

（本章完）
