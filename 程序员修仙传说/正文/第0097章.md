# 第97章 拒绝拜师

成为真传弟子的消息，很快传遍了整个碧寒宫。

一时间，我成了宗门内最热门的话题。

走到哪儿都有人指指点点，甚至还有大胆的女弟子直接上来表白。

"尤溪师兄，我喜欢你！"

一个面容清秀的女弟子红着脸，挡在我面前。

我："..."

这都第几个了？

"那个...师妹啊，"我尴尬地挠挠头，"咱们好像不太熟吧？"

"感情可以慢慢培养嘛！"女弟子鼓起勇气说道，"尤溪师兄，给我一个机会好不好？"

看着她期待的眼神，我心中叹了口气。

这要是在前世，有美女主动表白，我肯定高兴得跳起来。

但现在...

"抱歉，"我认真地说道，"我心中已有大道，暂不考虑儿女私情。"

女弟子眼中闪过一抹失望，但还是倔强地说道："那我等师兄！"

说完，她转身跑开了。

我无奈地摇摇头，继续向住处走去。

...

基础辅助：我最近的桃花运是不是有点旺？"

"根据数据分析，使用者近三天内收到表白17次，送花32次，送情书8封。"

"桃花运指数：99.9分。"

【建议使用者：保持冷静，专心修炼。"

我翻了个白眼。

专心修炼说得容易，但每天被人围观，怎么专心？

"尤溪！"

正想着，石鸾的声音从身后传来。

我转头看去，只见她急匆匆地跑过来，脸上带着一丝凝重。

"怎么了？"我问道。

"尚琼易来了。"石鸾低声说道。

"什么？！"

我瞳孔一缩。

这老东西，竟然追到碧寒宫来了？

...

碧寒宫大殿。

当我赶到时，掌门和各位长老已经在场了。

而尚琼易...

正坐在客位上，面无表情地品着茶。

虽然他看起来有些虚弱，但那股金丹巅峰的气势依然让人不敢小觑。

"尚盟主大驾光临，不知有何贵干？"掌门客气地问道。

尚琼易放下茶杯，目光落在我身上。

那一瞬间，我感觉到一股无形的压力扑面而来。

"本座此来，是为了尤溪。"他淡淡地说道。

"尤溪？"掌门眉头一皱，"尚盟主此话何意？"

尚琼易站起身，走到我面前。

他比我高半个头，居高临下地看着我，眼中神色复杂。

"尤溪，"他缓缓开口，"本座想正式收你为亲传弟子。"

这话一出，全场哗然！

亲传弟子？！

不是记名弟子，而是亲传弟子！

这可是师徒关系中最亲密的一种！

一旦成为亲传弟子，就能继承师父的一切，包括功法、法宝、甚至宗门地位！

尚琼易作为修仙盟的盟主，收亲传弟子...

这是何等荣耀的事情！

掌门和长老们的脸色都变了。

如果我真的成为尚琼易的亲传弟子...

那碧寒宫就失去了我这个天才！

"尚盟主，"掌门沉声说道，"尤溪是我碧寒宫的弟子，此举恐怕不妥吧？"

尚琼易淡淡地看了他一眼："掌门此言差矣。"

"尤溪本来就是本座的记名弟子，现在只是晋升为亲传而已。"

"而且..."他嘴角勾起一抹意味深长的笑容，"成为本座的亲传弟子，对他的前途只有好处，没有坏处。"

"碧寒宫虽强，但比起修仙盟，还是差了一些吧？"

这话一出，掌门脸色一沉。

尚琼易说得没错。

修仙盟是修仙界最强大的势力，汇集了各大宗门的精英。

碧寒宫虽然在北域称雄，但和修仙盟相比，确实差了一筹。

如果尚琼易真心收我为徒...

这确实是一个天大的机缘。

但问题是...

我信不过这老东西！

...

"尚盟主，"我深吸一口气，开口说道，"多谢您的厚爱，但请恕晚辈不能答应。"

尚琼易眉头一皱："为何？"

"因为..."我看着他的眼睛，一字一句地说道，"晚辈已有传承，不便再拜他人为师。"

"已有传承？"

尚琼易眼中闪过一抹寒光："什么传承？"

"虚空宗的传承。"我说道。

这话一出，全场寂静。

虚空宗！

那个传说中的上古宗门！

虽然早已覆灭，但它的传承...

依然是修仙界最顶级的传承之一！

"你...得到了虚空宗的传承？"尚琼易的声音有些颤抖。

"是的。"我点点头，"所以晚辈已经有了师承，不能再拜您为师了。"

"请尚盟主见谅。"

...

沉默。

死一般的沉默。

尚琼易看着我，眼中的神色从震惊变成了阴沉，最后...

变成了一抹难以察觉的杀意。

但他很快就掩饰了过去，换上了一副大度的表情。

"呵呵，原来如此。"他笑了笑，"虚空宗的传承...确实比本座的传承更强。"

"尤溪，你做出了正确的选择。"

"本座...尊重你的选择。"

他说着，转身向殿外走去。

但就在他经过我身边时...

我听到了一声微不可察的冷哼。

"哼，不识抬举的东西..."

声音很轻，只有我一个人能听见。

我心中一凛。

果然...

这老东西表面大度，实际上已经恨我入骨了！

...

尚琼易离开后，大殿内的气氛依然凝重。

"尤溪，"掌门看着我，眼中带着担忧，"你拒绝尚琼易，恐怕会惹来大麻烦。"

"弟子知道。"我说道，"但弟子更知道，如果答应了他，麻烦会更大。"

"哦？为何？"

"因为..."我沉声说道，"尚琼易收我为徒，根本不是为了培养我，而是为了图谋我的秘密。"

"虚空宗的传承、混沌空间金丹...这些都是他想要的。"

"一旦我真的成为他的亲传弟子，就相当于羊入虎口，任人宰割。"

掌门和长老们面面相觑。

"你说的有道理..."掌门点点头，"但尚琼易此人心胸狭窄，你当众拒绝他，他肯定不会善罢甘休。"

"接下来，你要格外小心。"

"弟子明白。"我躬身说道。

...

离开大殿后，我和石鸾回到了住处。

"尤溪，你刚才太冒险了。"石鸾担忧地说道，"尚琼易那个眼神...真的很可怕。"

"我知道。"我叹了口气，"但如果不拒绝，我会更危险。"

"与其成为他的棋子，不如明着跟他撕破脸。"

"至少现在，他不敢明着来。"

石鸾点点头："那你接下来怎么办？"

"修炼。"我说道，"尽快提升实力。"

"只要我有足够的实力，就不用怕任何人！"

"嗯，我陪你一起！"

...

当天晚上，我开始闭关修炼。

混沌空间金丹的力量浩瀚如海，我需要时间来熟悉和掌控。

"废物！都是废物！"

尚琼易怒吼道，一掌将面前的石桌拍成粉碎。

"盟主息怒..."一个黑衣人战战兢兢地说道。

"息怒？"尚琼易冷笑，"那小子当众拒绝本座，让本座颜面尽失，你让本座怎么息怒？！"

"可是...他现在有碧寒宫护着，我们不好下手啊..."

"碧寒宫？"尚琼易眼中闪过一抹阴毒，"碧寒宫护得了他一时，护不了他一世！"

"传令下去，启动'猎鹰计划'。"

"是！"

黑衣人领命而去。

尚琼易站在山谷中，望着碧寒宫的方向，眼中满是怨毒。

"尤溪...你以为拒绝了本座就安全了吗？"

"太天真了..."

"本座想要的东西，从来没有得不到的！"

"虚空宗的传承、混沌空间金丹...这些迟早都是本座的！"

"而你..."

他嘴角勾起一抹残忍的笑容。

"会为今天的拒绝，付出惨痛的代价！"
