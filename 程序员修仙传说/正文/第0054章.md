# 第54章 凡人城市

青云城内，人声鼎沸。

街道两旁店铺林立，叫卖声此起彼伏。

有卖灵草的药铺，有卖法器的铁匠铺，还有卖灵食的饭馆...

虽然都是些低阶物品，但对于凡人来说，已经算得上"仙家宝物"了。

"好热闹..."

石鸾看着眼前的景象，眼中带着好奇。

她从小就在碧寒宫长大，很少来凡间。

这种烟火气，让她感到新鲜又陌生。

"师姐，这就是凡间。"

尤溪笑着解释道。

"修仙者追求长生大道，但凡人也有自己的生活。"

"他们虽然寿命短暂，但活得同样精彩。"

...

石鸾若有所思地点点头。

确实...

凡人的生命虽然短暂，但他们有家人、有朋友、有喜怒哀乐...

这种平凡的幸福，或许也是修仙者所追求的终极境界吧？

"Potato，开始收集数据！"

"收到！"

"凡间数据收集中..."

"1. 城市人口密度：约1000人/平方公里"

"2. 灵气浓度：比宗门低80%，但比野外高20%"

"3. 凡人生活方式：以农业和手工业为主，商业发达"

"4. 修仙者渗透度：约0.001%，即每万人中有1个修仙者"

"5. 凡人对修仙者的态度：敬畏、向往、依赖"

...

尤溪一边走，一边观察着周围的一切。

他发现...

凡人对修仙者的依赖，比他想象的还要深！

街边的药铺，卖的是修仙者炼制的低阶丹药。

铁匠铺打造的武器，掺入了修仙者提供的灵铁。

甚至连饭馆里的灵食，也是用修仙者种植的灵米做成。

"基础辅助分析凡人对修仙者的依赖程度。"

【分析中..."

"凡人对修仙者的依赖度：73.6%"

"主要依赖领域：医疗、安全、生产、娱乐"

"结论：修仙者已经成为凡间社会的重要组成部分！"

...

尤溪心中一动。

这个发现很重要！

修仙者不是高高在上的存在...

他们与凡间，其实是相互依存的关系！

"如果能建立凡间观察网络，不仅能收集数据，还能..."

"深入了解这种依存关系！"

...

两人继续前行，来到城市的中心广场。

广场上人头攒动，似乎在看什么热闹。

尤溪好奇地挤进去...

只见广场中央，搭着一个高台。

高台上，一个身穿道袍的中年人正在表演"仙术"。

"各位父老乡亲，请看！"

那道士手一挥，一道火焰凭空出现！

"哇——！！"

围观的凡人发出阵阵惊呼！

"这是仙术！真正的仙术！"

"道长真是活神仙啊！"

...

尤溪看着这一幕，嘴角抽搐。

这哪里是什么仙术...

分明是最低级的火球术！

而且这个道士...

修为最多炼气三层！

"基础辅助分析那个道士。"

"收到！"

"目标分析："

"修为：炼气三层"

"功法：低阶火属性功法"

"状态：依靠表演仙术骗取钱财"

"威胁度：极低"

...

原来是个骗子！

尤溪摇摇头。

修仙界就是这样...

有真本事的，往往低调内敛。

没本事的，反而招摇撞骗。

...

就在这时——

那道士的目光突然落在尤溪身上。

他的眼中闪过一抹贪婪。

"这位小兄弟，我看你骨骼清奇，是个修仙的好苗子！"

道士跳下高台，走到尤溪面前。

"要不要拜我为师？我教你修仙之术！"

...

尤溪："？？？"

这骗子...

骗到他头上了？

"不好意思，没兴趣。"

尤溪淡淡地说道，拉着石鸾准备离开。

但道士却不依不饶。

"别走啊！"

"我看你体内有灵根波动，是块修仙的好材料！"

"错过我，你会后悔一辈子的！"

...

尤溪停下脚步，似笑非笑地看着他。

"你说我体内有灵根波动？"

"那你倒是说说，我是什么灵根？"

...

道士一愣。

他哪里看得出尤溪是什么灵根？

刚才那句话，不过是他的惯用套路罢了！

"这个...这个嘛..."

"你是...你是..."

他支支吾吾，说不出个所以然。

...

尤溪冷笑一声。

"连自己是什么修为都看不出来，还想收徒弟？"

话音未落——

他释放出一丝气息！

筑基中期巅峰的威压，瞬间笼罩全场！

...

"扑通！"

道士直接跪在了地上！

他的脸色惨白，浑身颤抖！

"前...前辈...小的有眼不识泰山..."

"求前辈饶命..."

...

围观的凡人也都被这股威压吓得跪倒在地！

他们虽然感受不到灵力波动...

但那种来自灵魂深处的恐惧，让他们本能地臣服！

"这就是...真正的仙人吗..."

"太可怕了..."

"刚才那个道士是骗子吧？"

"真正的仙人一出，他就露馅了..."

...

尤溪收回威压，淡淡地说道：

"滚吧。"

"以后不要再招摇撞骗。"

"否则..."

他眼中闪过一抹寒光。

"下次就没这么好运了。"

...

"是是是！小的明白！小的这就滚！"

道士连滚带爬地逃离了广场。

...

尤溪摇摇头，拉着石鸾继续往前走。

但周围的凡人却将他们团团围住！

"仙人！请收我为徒吧！"

"仙人，求您赐我一粒仙丹！"

"仙人，我家有良田百亩，愿全部献给仙人！"

...

尤溪被吵得头疼。

他没想到，只是释放了一丝气息，就引起这么大的轰动！

"安静！"

他一声低喝，全场安静下来。

"我不是什么仙人，只是一个普通的修仙者。"

"我不会收徒，也不会赐丹。"

"但我可以告诉你们..."

他环视四周，沉声说道：

"修仙之路，凶险万分。"

"若无大毅力、大勇气、大智慧，绝难成功。"

"你们若想修仙，就去寻找真正的仙门，不要轻信这些江湖骗子！"

...

说完，他带着石鸾，身形一闪，消失在原地。

...

凡人面面相觑，然后爆发出热烈的议论声。

"刚才那位...才是真正的仙人啊..."

"太厉害了，来无影去无踪！"

"我要修仙！我一定要修仙！"

...

尤溪和石鸾来到城市边缘的一座高塔上。

"呼...终于清静了。"

尤溪松了口气。

"师姐，你没事吧？"

石鸾摇摇头。

"没事，只是没想到...凡人对修仙者的热情这么高。"

"那是当然。"

尤溪笑道。

"长生不死，谁不想呢？"

...

他站在高塔上，俯瞰整座城市。

"Potato，找到合适的节点位置了吗？"

【分析中..."

"推荐位置：高塔地下密室"

"理由："

"1. 位置隐蔽，不易被发现"

"2. 地下有微弱灵脉，可为阵法提供能量"

"3. 视野开阔，便于监测全城"

...

尤溪点点头。

"好，就去那里！"

...

两人来到高塔地下，果然发现了一个隐秘的密室。

密室不大，但很干净，似乎很久没人来过了。

"就是这里了。"

尤溪取出阵盘，开始布置观察节点。

"Potato，开始记录阵法数据！"

"收到！"

"凡间第一个观察节点..."

"开始建设！"

...

与此同时。

城市另一边。

那个跟踪尤溪的黑影，终于发现自己被耍了。

"该死！"

"竟然跟丢了！"

他咬牙切齿，取出传音符。

"少主，目标跟丢了..."

"废物！"

传音符那头，传来温如玉愤怒的咆哮。

"继续找！"

"找到他，杀了他！"

...

黑影收起传音符，眼中闪过一抹杀意。

"尤溪..."

"你以为逃得掉吗？"

"血影楼的杀手，从来不留活口！"

（本章完）
