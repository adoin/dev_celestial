# 第98章 返回碧寒宫

魔门异动，整个碧寒宫都笼罩在一片紧张的气氛中。

弟子们加强了巡逻，护山大阵全天开启，连平日里热闹的练武场都变得冷冷清清。

"这气氛，跟以前在公司赶项目deadline的时候一样压抑啊..."我走在宗门的小道上，忍不住吐槽。

"尤溪师兄！"

一个年轻弟子跑过来，恭敬地行礼："掌门召您去大殿议事。"

"知道了。"

我整理了一下衣衫，向大殿走去。

...

大殿内，除了掌门和各位长老外，还有几个陌生的面孔。

他们身穿各异的服饰，显然是其他宗门的代表。

"尤溪，你来了。"掌门点点头，"坐下吧。"

我找了个位置坐下，打量着周围的人。

"介绍一下，"掌门说道，"这几位是北域其他宗门的使者。"

"魔门此次来势汹汹，我们决定联合抗敌。"

"尤溪，你现在是碧寒宫的真传弟子，地位等同于长老，有资格参与此次议事。"

我点点头，心中却有些感慨。

一个月前，我还是个小透明。

现在...

却能坐在大殿里，参与宗门最高级别的议事。

这就是实力的重要性啊。

...

"诸位，"掌门开口说道，"根据探子回报，魔门此次出动了三位魔将，率领数千魔兵，正在向北域进发。"

"三位魔将？！"一个使者惊呼道，"那可是相当于元婴期的存在！"

"没错。"掌门脸色凝重，"而且据可靠消息，这三位魔将中，有一位是血魔将。"

"血魔将！"

众人倒吸一口凉气。

血魔将，魔门十二魔将之一，实力在元婴期中都算是顶尖的存在！

"这下麻烦了..."一个使者苦着脸说道，"我们北域最强的也就是掌门您，金丹巅峰..."

"对上血魔将，恐怕..."

他没有说下去，但意思已经很明显了。

"所以，"掌门沉声说道，"我们需要求援。"

"已经向修仙盟发出了求救信号，相信很快就会有援军到来。"

"在援军到来之前，我们必须守住防线！"

"诸位，可愿与碧寒宫并肩作战？"

"愿与碧寒宫共存亡！"众人齐声应道。

...

会议结束后，我被掌门单独留了下来。

"尤溪，"掌门看着我，眼中带着复杂的神色，"你觉得这次魔门来袭，正常吗？"

我心中一动："掌门也觉得有问题？"

"当然有问题。"掌门冷笑一声，"魔门虽然猖獗，但从未如此明目张胆地进攻北域。"

"而且时间也太巧了，正好在你击败尚琼易之后..."

"您怀疑..."我压低声音，"这是尚琼易的手笔？"

掌门摇摇头："尚琼易虽然心胸狭窄，但还不至于勾结魔门。"

"但...他可能会借机生事。"

"尤溪，你要小心。这次魔门来袭，可能有人会趁机对你下手。"

我点点头："弟子明白。"

...

离开大殿后，我回到了自己的新住处。

作为真传弟子，我有了一座独立的洞府，灵气浓度是普通弟子房间的十倍。

"这待遇，果然不一样啊..."我躺在软榻上，感慨道。

"尤溪！"石鸾走了进来，"听说魔门出动了三位魔将？"

"是啊，"我坐起身，"情况有点严峻。"

"那你..."石鸾担忧地看着我，"会有危险吗？"

"危险肯定是有的。"我说道，"但这也是机会。"

"机会？"

"嗯。"我眼中闪过一抹精光，"魔门之人的身上，往往有特殊的魔核。"

"那些魔核对我的混沌空间金丹，有极大的滋补作用。"

"如果能猎杀几头魔将..."

"我的修为说不定能突破到金丹后期！"

石鸾瞪大眼睛："你疯了吗？那可是元婴期的存在！"

"现在打不过，不代表以后打不过。"我笑道，"而且我有Potato辅助，打不过也可以跑嘛。"

"你..."石鸾无奈地摇摇头，"真拿你没办法。"

"不过..."她认真地看着我，"如果你要去，一定要带上我。"

"咱们说好了，并肩作战的。"

我心中一暖，笑着点点头："好，一起！"

...

接下来的几天，整个碧寒宫都处于紧张的备战状态。

弟子们日夜操练，阵法师们加固护山大阵，炼丹师们炼制各种丹药...

而我...

则在洞府中闭关修炼。

"Potato，智能体2.0的开发进度如何？"

"当前进度：85%。"

"核心算法已优化完成，正在进行最后的测试。"

"预计三天后，可以正式投入使用。"

三天...

我点点头。

智能体2.0是我结合虚空宗传承和现代AI技术开发的新一代辅助程序。

它不仅可以辅助修炼，还可以...

预测敌人的攻击！

这是我在与尚琼易战斗时得到的灵感。

如果能提前预知敌人的攻击轨迹，那在战斗中将占据绝对的优势！

"加快进度，争取在魔门到来之前完成。"

"收到。正在分配更多计算资源..."

...

三天后。

"叮！"

"智能体2.0开发完成！"

"所有模块运行正常，可以投入使用！"

我睁开眼睛，眼中闪过一抹兴奋。

终于完成了！

"Potato，测试一下新功能。"

"收到。启动战斗预测模块..."

"请使用者进行模拟战斗。"

我站起身，在洞府中演练起一套剑法。

剑光闪烁，身影飘忽。

"分析中..."

"检测到使用者下一招有87%概率使用'回风拂柳'，13%概率使用'长虹贯日'。"

"建议应对方式：提前封住左侧空档。"

我点点头。

预测准确率还不错。

"继续测试。"

我加快了攻击速度，招式变得更加复杂多变。

但Potato依然能够准确预测我的下一步动作。

预测准确率：92%！

"牛逼！"我忍不住赞叹。

有了这个，我在战斗中的胜算将大大提升！

"尤溪！"

就在这时，石鸾急匆匆地跑了进来。

"怎么了？"我问道。

"魔门的人...来了！"

...

碧寒宫外，黑云压城。

数千魔兵整齐列阵，散发着滔天的魔气。

而在魔兵前方，三道身影凌空而立。

正是魔门的三位魔将！

为首的是一个身穿血袍的中年男子，面容英俊却透着一股邪异之气。

血魔将！

"碧寒宫的人听着！"血魔将开口说道，声音如同滚雷，"交出尤溪，饶你们不死！"

这话一出，碧寒宫上下一片哗然。

"冲着我来的？"我眉头一皱。

"尤溪，别出去！"石鸾拉住我的手，"他们明显是冲着你来的！"

"我知道。"我说道，"但我不能躲。"

"如果躲了，碧寒宫就会因为我而遭受攻击。"

"这不是我的风格。"

我挣脱石鸾的手，向阵外飞去。

"尤溪！"石鸾大喊，但也只能跟了上来。

...

阵外，我凌空而立，与血魔将遥遥相对。

"你就是尤溪？"血魔将打量着我，眼中带着一丝兴趣。

"凝结混沌空间金丹的小子？"

"正是在下。"我说道，"不知血魔将找我，有何贵干？"

"呵呵..."血魔将笑了笑，"有人花大价钱，买你的命。"

"哦？"我眉头一挑，"谁这么大方？"

"这个嘛..."血魔将眼中闪过一抹戏谑，"等你死了，我会告诉你的。"

"上！"

一声令下，数十个魔兵向我扑来！

"来得好！"

我冷笑一声，催动混沌空间金丹。

"空间领域·展开！"

银白色的领域瞬间扩散，将那些魔兵笼罩其中。

"这是什么？！"魔兵们大惊失色。

在空间中，他们的行动变得迟缓，仿佛陷入了泥沼。

"五行空间·斩！"

我挥出一剑，银白色的剑芒横扫而出！

噗噗噗！

那些魔兵根本来不及反应，就被斩成了两半！

"有点意思。"血魔将眼中闪过一抹惊讶，"难怪有人要花大价钱买你的命。"

"你确实有些本事。"

"不过..."他冷笑道，"在本将面前，你还是太嫩了！"

他说着，一掌拍出！

一只巨大的血色手掌凭空出现，向我拍来！

那手掌散发着滔天的魔气，所过之处，连空间都在扭曲！

"Potato！"

"检测到元婴期攻击！启动战斗预测模块！"

"分析中...攻击轨迹已锁定！"

"建议：向左侧闪避15米，同时催动空间护盾！"

我毫不犹豫地照做！

嗖！

我的身形向左一闪，同时凝聚出一面空间护盾。

轰！！！

血色手掌擦着我的身体拍过，被空间护盾挡住了一部分威力。

虽然还是被余波震得气血翻涌，但...

我活下来了！

"什么？！"血魔将瞪大眼睛，"你竟然躲过了本将的攻击？！"

"运气好而已。"我擦了擦嘴角的血迹，笑道。

但心中却是一片振奋。

智能体2.0...

真的能预测元婴期的攻击！

"找死！"

血魔将大怒，再次出手！

这一次，他的攻击更加狂暴，几乎封锁了我所有的退路！

"Potato！"

"警告！攻击覆盖范围过大，无法完全躲避！"

"建议：硬抗一部分，利用空间核心转移伤害！"

"好！"

我咬紧牙关，准备迎接这一击。

...

就在这时——

"住手！"

一声暴喝从天际传来！

一道流光破空而至，挡在了我面前。

轰！！！

血魔将的攻击被那道流光挡住，化作漫天的灵气风暴。

当风暴散去时...

一个熟悉的身影出现在我面前。

"刘长老？！"我惊讶道。

"不...这股气息..."

"元婴期？！"

刘长老回头看了我一眼，笑道："尤溪，做得不错。"

"接下来的事情，交给我吧。"

我心中一震。

刘长老...

竟然突破到元婴期了？！

...

"元婴期？"血魔将眯起眼睛，"碧寒宫竟然还藏着这样的高手？"

"哼，血魔将，你的对手是我！"刘长老冷哼道。

"老家伙，你找死！"

血魔将大怒，与刘长老战在一起！

两个元婴期强者的战斗...

那场面，简直惊天动地！

"尤溪，快回去！"石鸾趁机拉着我退回了护山大阵内。

"刘长老他..."我担心地说道。

"放心吧，"掌门不知何时出现在我身边，"刘长老刚突破元婴期，正需要一场大战来稳固境界。"

"而且..."

他看向远方，眼中闪过一抹精光。

"援军，马上就到了。"

...

果然，没过多久。

天际出现了一片流光。

修仙盟的援军，到了！

为首的是一个身穿紫袍的老者，气息之强，甚至还在血魔将之上！

"紫袍尊者！"血魔将脸色大变，"撤！"

他当机立断，下令撤退。

数千魔兵如潮水般退去，转眼间就消失在天际。

"想走？"紫袍尊者冷哼一声，"留下点代价吧！"

他一掌拍出，将一位魔将当场击杀！

剩下的两位魔将（包括血魔将）拼命逃窜，才勉强逃过一劫。

...

战斗结束，碧寒宫上下欢呼雀跃。

而我...

却在想一个问题。

血魔将说的那个"花大价钱买我命"的人...

到底是谁？

"Potato，分析一下。"

"根据当前情报分析，嫌疑人有三个："

"1. 尚琼易（概率：65%）"

"2. 温如玉（概率：25%）"

"3. 其他未知势力（概率：10%）"

我点点头。

果然...

尚琼易的可能性最大。

"看来，我得主动出击了。"

我眼中闪过一抹寒光。

被动挨打不是我的风格。

既然敌人已经出手...

那就别怪我不客气了！

（本章完）
