# 第41章 追杀再起

"你疯了吗？！"

石鸾惊呼。

"他们可是有筑基大圆满！"

"正面对上，我们必死无疑！"

尤溪却咧嘴一笑。

"师姐，你忘了？"

"我可是程序员。"

"程序员的第一原则是什么？"

"能跑就跑，能苟就苟？"

"错！"

尤溪摇头。

"程序员的第一原则是——利用一切可以利用的资源，以最小的代价解决最大的问题！"

"他们人多势众，但有一个致命的弱点。"

"什么弱点？"

"他们太分散了。"

尤溪解释道。

"扇形搜索，虽然能覆盖更大的范围，但也意味着每个人之间的距离都很远。"

"我们不需要对付所有人。"

"只需要...

"逐个击破！"

...

树林中。

一个筑基中期的修士正在谨慎地搜索。

他是温如玉派来的二十名追杀者之一。

"那小子到底跑哪去了..."

他喃喃自语。

"该死的寂云谷，神识被压制得厉害，根本探查不了多远..."

就在这时——

"嗖！"

一道破空声从身后传来！

他猛地转身，却什么都没看到。

"什么？"

他愣了一下。

"在后面哦。"

一个戏谑的声音从背后响起。

他惊恐地回头——

"噗！"

一道金刃贯穿了他的咽喉！

尤溪的身影出现在他身后，面无表情地抽出金刃。

"第一个。"

他轻声说道，然后消失在树林中。

...

"Potato，下一个目标？"

"左前方300米，筑基初期。"

"收到。"

尤溪如同幽灵般在树林中穿梭。

他的身形飘忽不定，时而出现在树梢，时而潜伏在灌木丛中。

Potato的实时导航，让他能完美避开敌人的感知，从死角发动攻击。

这是...刺客的打法！

...

第二个、第三个、第四个...

短短半个时辰，尤溪已经悄无声息地干掉了八名追杀者！

而且都是一击必杀，没有发出任何声响！

...

"不对劲..."

追杀队伍的中心，一个身穿黑袍的青年皱起眉头。

他是这次行动的领队，筑基大圆满修为，名叫赵无极。

"其他人呢？"

"怎么联系不上了？"

他取出传音符，连续呼叫了几个人，都没有回应。

"该死...出事了！"

赵无极脸色阴沉。

"所有人，向我靠拢！"

"停止搜索，保持警惕！"

但他的命令已经晚了。

...

树林边缘。

尤溪躲在一片灌木丛后，观察着敌人的动向。

"Potato，还剩多少人？"

"剩余敌人：12人。"

"其中筑基初期3人，筑基中期6人，筑基后期2人，筑基大圆满1人。"

"敌人已经开始收缩阵型，偷袭难度大幅提升。"

尤溪点点头。

剩下的都是硬茬子了。

尤其是那个筑基大圆满，不是现在的他能对付的。

"得想个办法，把他们分开..."

他思索着，目光落在了背上的石鸾身上。

"师姐，我有个计划。"

"什么计划？"

"需要你做诱饵。"

石鸾："...你是不是想死？"

"咳，听我说完。"

尤溪连忙解释。

"你在这里现身，假装受伤逃窜，吸引他们的注意力。"

"我从侧面绕过去，干掉那些筑基中期和初期的。"

"等解决掉杂鱼，我们再合力对付那个筑基大圆满。"

石鸾沉默了一会。

"你确定这样能行？"

"七成把握。"

"...好。"

石鸾深吸一口气。

"我信你。"

她从尤溪背上下来，活动了一下身体。

"但要是你搞砸了，我做鬼也不会放过你。"

尤溪笑道：

"放心，我可是专业的。"

...

计划开始。

石鸾故意弄出动静，从树林中"逃"出来。

她浑身是血（其实是尤溪之前留下的），脸色苍白，看起来狼狈不堪。

"在那边！"

"发现目标了！"

追杀者立刻发现了她，纷纷追了上来。

"快追！别让她跑了！"

赵无极冷哼一声。

"不用追。"

他取出一把黑色的长弓，搭箭拉弦。

"嗖！"

一支黑色的箭矢破空而出，直奔石鸾的后心！

这一箭速度极快，威力惊人！

石鸾感觉到背后的危险，连忙闪避。

"嗤！"

箭矢擦过她的肩膀，带起一蓬鲜血！

"呃！"

石鸾闷哼一声，摔倒在地。

"抓住她！"

赵无极下令。

几个筑基中期的修士冲了上去。

...

就是现在！

尤溪从侧面的树林中冲出！

"空间禁锢！"

他双手结印，五行灵力爆发！

冲在最前面的三个筑基中期修士瞬间被定在原地！

"什么？！"

他们惊恐地发现，自己动不了了！

"死！"

尤溪手中金刃连闪！

"噗噗噗！"

三颗人头落地！

"有埋伏！"

其他人大惊失色。

但尤溪已经消失在原地，出现在另一个方向！

"空间禁锢！"

又是两个人被定住！

"噗噗！"

又是两条人命！

...

赵无极脸色铁青。

"该死！"

"一起上！围住他！"

剩下的七个人同时扑向尤溪。

但尤溪的身法太诡异了！

他仿佛能预判所有人的攻击，每次都能从不可能的角度闪避，然后发动致命反击！

"Potato，计算所有攻击轨迹！"

"计算完成！"

"左前方，火球术，0.3秒后到达，建议向左闪避0.5米。"

"右后方，剑气，0.5秒后到达，建议向前翻滚。"

"头顶，刀光，0.8秒后到达，建议..."

尤溪按照Potato的指示，在敌人的围攻中游刃有余地穿梭。

每一次闪避都恰到好处，每一次反击都精准致命！

...

"这小子...邪门！"

赵无极越看越心惊。

尤溪明明只有筑基初期的修为，但战斗力却远超境界！

那种诡异的身法，那种精准的预判...

简直就像...提前知道所有人的攻击一样！

"不能再拖了！"

赵无极一咬牙，亲自出手！

"小子，受死！"

他手中出现一把黑色的长刀，刀身上缠绕着浓郁的黑气。

"黑风刀法·斩魂！"

一道黑色的刀光破空而出，速度快到极致！

尤溪心中警铃大作！

这一刀...

他躲不开！

"Potato！"

"检测到致命攻击！"

"攻击速度超出闪避极限！"

"建议：硬抗或...用那个！"

那个？

尤溪瞬间明白了Potato的意思。

他新领悟的空间术法！

"空间折叠！"

尤溪双手结印，体内的五行灵力疯狂涌动！

刹那间——

他面前的空间发生扭曲，那道黑色刀光竟然...偏离了方向！

"什么？！"

赵无极瞪大了眼睛。

他的斩魂刀，竟然被...折射了？！

刀光从尤溪身侧掠过，斩断了旁边的一棵大树。

"这不可能！"

赵无极难以置信。

这是什么术法？！

竟然能改变攻击的轨迹？！

...

尤溪也愣住了。

他只是想试试空间折叠的效果，没想到竟然真的成功了！

"Potato，刚才那是..."

"空间折叠，利用五行之力扭曲局部空间，改变物体的运动轨迹。"

"这是《空间术法详解》中记载的高级技巧。"

"恭喜宿主，成功掌握空间折叠！"

尤溪狂喜！

有了这个能力，他就能正面与筑基大圆满抗衡了！

"再来！"

他主动冲向赵无极！

...

两人激战在一起。

赵无极的攻击凌厉无比，每一刀都带着撕裂空气的威力。

但尤溪总能利用空间折叠，将致命的攻击偏移。

同时，他的反击也越来越犀利。

金刃、火球、土刺...

各种术法层出不穷，让赵无极应接不暇。

"该死！该死！"

赵无极越打越心惊。

这小子...简直是个怪物！

明明修为不如他，却能与他打成平手！

再拖下去...

对自己不利！

"撤！"

赵无极咬牙做出决定。

"所有人，撤退！"

他转身就逃。

其他几个幸存者也跟着逃跑。

"想跑？"

尤溪冷笑。

"晚了！"

"空间禁锢！"

他全力催动灵力，将剩下的人全部定住！

"死！"

金刃闪过，人头落地。

转眼间，就只剩下赵无极一个人了。

...

赵无极脸色惨白，拼命奔跑。

但他发现，尤溪的速度竟然比他更快！

"该死...该死..."

他咬牙，从怀中取出一枚红色的丹药，吞入腹中。

"燃血丹！"

他的气息瞬间暴涨，速度提升了一倍！

这是燃烧生命本源的禁药，代价极大，但效果惊人。

"小子，这次算你运气好！"

"下次见面，我必取你性命！"

他的身影迅速远去，消失在树林深处。

尤溪追了一段，不得不放弃。

燃血丹的效果太强，他追不上。

"让他跑了..."

他皱起眉头。

赵无极逃回去，肯定会向温如玉汇报。

接下来，等待他的将是更猛烈的追杀！

...

"尤溪！"

石鸾跑了过来，脸上满是担忧。

"你没事吧？"

"没事。"

尤溪摇摇头。

"可惜让那个领头的跑了。"

"你已经很厉害了..."

石鸾看着满地的尸体，心中震撼。

二十名追杀者，被尤溪一个人干掉了十九个！

其中包括十几个筑基中期，几个筑基后期！

这战绩...

简直骇人听闻！

"我们得赶紧离开这里。"

尤溪说道。

"刚才的战斗动静太大，可能会引来妖兽或者其他追杀者。"

"而且..."

他看向远处。

"我感觉到，有一股更强的气息正在接近。"

"那股气息...比筑基大圆满还要恐怖！"

石鸾脸色一变。

"金丹期？！"

"不清楚。"

尤溪摇头。

"但不管是谁，现在的我们对上，都没有胜算。"

"走！"

他拉起石鸾的手，快速离开战场。

...

就在他们离开后不久。

一道身影出现在战场上。

那是一个身穿青袍的老者，面色阴鸷，气息恐怖。

如果尤溪在这里，一定会认出来——

这是青牛真人！

金丹期强者！

"有趣..."

青牛真人看着满地的尸体，眼中闪过一丝玩味。

"以筑基初期，击杀近二十名同阶甚至高阶修士..."

"这小子，身上果然有大秘密！"

他闭上眼睛，神识如潮水般向四周扩散。

金丹期的神识，覆盖范围可达数十里！

"找到了..."

他嘴角勾起一抹冷笑。

"小子，你逃不掉的。"

他的身影一闪，消失在原地。

...

远处。

尤溪突然感觉背后一凉。

"Potato，怎么了？"

"警告！"

"检测到极强灵压正在快速接近！"

"等级：金丹期！"

"距离：5000米...4500米...4000米..."

"正在以极快的速度逼近！"

尤溪脸色大变。

金丹期？！

怎么会有金丹期强者追杀他？！

"师姐，快跑！"

"有金丹期追来了！"

石鸾也吓得花容失色。

金丹期！

那可是宗门长老级别的存在！

两人拼命奔跑，但那股恐怖的灵压越来越近...

越来越近...

（本章完）
