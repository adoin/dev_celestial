# 第58章 异常数据

眼前一花。

尤溪只觉得天旋地转...

等他再次睁开眼睛时...

已经身处一个陌生的山洞中！

山洞不大，但收拾得很干净。

洞壁上镶嵌着发光的灵石，将整个山洞照得通明。

石鸾躺在一旁的石床上，气息平稳，似乎只是昏睡过去了。

...

而在洞口...

青牛真人负手而立，背对着他。

那股恐怖的威压...

让整个山洞的空气都仿佛凝固了！

"醒了？"

青牛真人淡淡开口。

他的声音不大...

却仿佛直接在尤溪脑海中响起！

...

尤溪深吸一口气，强迫自己冷静下来。

"前辈...不知带晚辈来此，有何贵干？"

...

青牛真人缓缓转身。

他的面容看起来很普通...

就像一个普通的老农。

但那双眼睛...

却深邃如星空！

仿佛能看穿一切！

"小子，你知道本座是谁吗？"

...

尤溪恭敬地说道：

"晚辈知道。"

"青牛真人，金丹期强者，威震方圆千里！"

...

青牛真人微微点头。

"既然知道..."

"那你应该也明白..."

"本座若要杀你，易如反掌！"

...

话音落下...

一股恐怖的威压...

瞬间笼罩尤溪！

"轰——！"

尤溪只感觉一座大山压在自己身上！

双腿一软...

差点跪倒在地！

"前辈...！"

他咬牙支撑，额头上青筋暴起！

...

青牛真人看着他挣扎的样子，眼中闪过一抹玩味。

"不错。"

"筑基中期，竟能在本座的威压下不跪..."

"有点意思。"

...

他收回威压，淡淡地说道：

"小子，本座问你几个问题。"

"答得好，饶你不死。"

"答不好..."

他眼中闪过一抹寒光。

"你就永远留在这里吧！"

...

尤溪心中一凛！

这就是金丹期的恐怖！

生杀予夺，一念之间！

"前辈请说...晚辈知无不言！"

...

青牛真人盯着他，缓缓问道：

"第一个问题..."

"你身上的空间波动，是怎么回事？"

...

尤溪心中一震！

果然...

是为了空间术法！

"回前辈..."

"晚辈偶然得到一部空间系功法，修炼所得..."

...

"偶然得到？"

青牛真人冷笑。

"什么功法？"

"从哪里得到的？"

...

尤溪犹豫了一下。

"功法名为《虚空经》..."

"是在一处上古遗迹中偶然所得..."

...

"《虚空经》？！"

青牛真人瞳孔一缩！

眼中闪过一抹狂喜！

"果然是空间系功法！"

"小子，把功法交出来！"

...

尤溪心中一沉！

果然...

是冲着功法来的！

"前辈...这功法是晚辈的机缘..."

...

"机缘？"

青牛真人冷笑。

"在本座面前..."

"你的命都是本座的！"

"更别说一部功法了！"

...

他一步踏出！

瞬间出现在尤溪面前！

速度快到尤溪根本反应不过来！

"交出功法！"

"否则...死！"

...

尤溪咬牙。

交出功法？

不行！

那是他最大的底牌！

但如果不交...

今天怕是走不出这个山洞了！

"Potato！分析当前情况！"

【分析中..."

"目标：青牛真人"

"修为：金丹初期"

"战力差距：约100倍"

"

"结论：不可力敌！"

【建议：暂时妥协，保留核心算法！"

...

尤溪心中一动。

保留核心算法？

对了！

《虚空经》分为上下两篇！

上篇是空间感应和基础运用...

下篇才是核心算法——空间折叠的详细原理！

他可以交上篇...

保留下篇！

"前辈..."

尤溪深吸一口气。

"晚辈可以交出功法..."

"但前辈能否保证...放晚辈离开？"

...

青牛真人眼中闪过一抹喜色！

"可以！"

"本座以道心起誓！"

"只要你交出完整的《虚空经》..."

"本座绝不为难你！"

...

尤溪点点头。

"好！"

他取出一枚空白玉简...

开始拓印功法！

当然...

他只拓印了上篇！

...

片刻后。

尤溪将玉简递给青牛真人。

"前辈，这是《虚空经》上篇..."

"晚辈只得到了上篇，下篇...晚辈也不知道在哪里..."

...

青牛真人接过玉简，神识一扫。

眼中顿时闪过一抹精光！

"果然是空间系功法！"

"虽然只是上篇..."

"但也足够本座参悟了！"

...

他满意地点点头。

"小子，你走吧。"

"本座说话算话！"

...

尤溪心中一松。

连忙抱起石鸾...

向洞外走去！

...

但就在他即将走出洞口时——

青牛真人的声音从背后传来。

"等等。"

...

尤溪脚步一顿！

心中一紧！

难道...

他发现了？！

"前辈...还有什么事？"

...

青牛真人淡淡地说道：

"你刚才布置的那些阵法...很有意思。"

"如果本座没猜错..."

"你应该是在收集数据吧？"

...

尤溪瞳孔一缩！

他连这个都知道？！

"晚辈...只是好奇..."

...

青牛真人摆摆手。

"本座不管你有什么目的。"

"但你要记住..."

他转过身，盯着尤溪。

"有些秘密..."

"知道得太多，会死人的！"

...

尤溪心中一凛！

"晚辈明白！"

...

说完，他不敢停留...

抱着石鸾，迅速离开！

...

等尤溪走后...

青牛真人看着手中的玉简，眼中闪过一抹阴冷。

"小子..."

"你以为本座看不出来？"

"这功法...根本不完整！"

...

他冷笑一声。

"不过没关系..."

"本座有的是时间..."

"让你乖乖交出完整的功法！"

...

...

尤溪抱着石鸾，一路狂奔！

直到跑出数百里...

才停下来喘息！

"呼...呼..."

"终于...逃出来了..."

...

他将石鸾放在一块平整的石头上...

检查她的伤势。

"Potato，检测师姐状态！"

"检测中..."

"目标生命体征：稳定"

"伤势：已大幅好转"

"预计苏醒时间：2小时内"

...

尤溪松了口气。

师姐没事就好...

...

他环顾四周...

发现自己身处一片密林中。

这里应该是青云岭的某个角落...

距离血影楼的分舵很远！

"Potato，重新连接观察节点！"

"连接中..."

"节点1：连接成功"

"节点2：连接成功"

"节点3：连接成功"

"节点4：连接成功"

"节点5：连接失败（已被摧毁）"

...

尤溪皱眉。

5号节点被摧毁了...

应该是血影楼的人干的！

"重新计算节点布局..."

"在青云岭补充两个节点！"

"收到！"

"正在重新规划..."

...

尤溪开始忙碌起来。

虽然刚从青牛真人手中逃脱...

但他不能停下脚步！

观察网络的建设...

必须继续！

...

两个时辰后。

石鸾缓缓睁开眼睛。

"师弟..."

...

尤溪连忙过来。

"师姐！你醒了！"

"感觉怎么样？"

...

石鸾坐起身，活动了一下身体。

"我没事..."

"师弟，我们这是在哪里？"

"血影楼的人呢？"

...

尤溪将事情的经过简单说了一遍...

当然...

隐瞒了青牛真人索要功法的事。

"...然后我们就逃到这里了。"

...

石鸾听完，眼中闪过一抹担忧。

"师弟...那青牛真人..."

"会不会再找上门来？"

...

尤溪摇头。

"暂时应该不会。"

"我给了他一部分功法...他应该会去参悟一段时间。"

...

石鸾松了口气。

"那就好..."

...

但就在这时——

Potato突然发出警报！

【警告！"

【检测到异常数据！"

...

尤溪一愣。

"什么异常数据？"

...

Potato快速汇报：

"根据1号节点数据..."

"青云城东门方向..."

【检测到异常空间波动！"

"

"波动特征：与青牛真人不同..."

"更古老...更强大！"

"

"位置：青云城外500里..."

"波动源：疑似上古遗迹！"

...

尤溪瞳孔一缩！

上古遗迹？！

空间波动？！

难道...

是《虚空经》的来源地？！

"Potato，详细分析！"

【分析中..."

"空间波动频率：0.037Hz"

"波动强度：中等"

"持续性：稳定"

"

"推测：上古遗迹禁制松动..."

"禁制类型：空间系！"

"

【警告：该遗迹可能与《虚空经》有关联！"

...

尤溪心中一动！

与《虚空经》有关？！

如果能找到遗迹...

说不定能找到《虚空经》的完整版本！

到时候...

就不用怕青牛真人的威胁了！

"师姐，我们有新目标了！"

...

石鸾疑惑地看着他。

"什么目标？"

...

尤溪眼中闪烁着兴奋的光芒！

"上古遗迹！"

"可能与我的空间功法有关的..."

"上古遗迹！"

...

石鸾惊讶地捂住嘴。

"上古遗迹？！"

"师弟，那可是危险重重的地方！"

"以我们的实力..."

...

尤溪笑道：

"所以才要去啊！"

"富贵险中求！"

"而且..."

他看向远方，眼中闪过一抹精光。

"如果不尽快提升实力..."

"等青牛真人发现我给的是残缺功法..."

"我们就真的死定了！"

...

石鸾沉默片刻...

然后坚定地点点头。

"好！"

"我陪你去！"

...

尤溪心中一暖。

"师姐..."

...

石鸾微微一笑。

"不管去哪里..."

"我都陪你。"

...

尤溪深吸一口气。

"那就出发！"

"目标——"

"上古遗迹！"

...

两人收拾妥当...

向青云城外飞去！

...

与此同时。

山洞中。

青牛真人正在参悟《虚空经》...突然眉头一皱！

"嗯？"

"这个空间波动是..."

他猛地睁开眼睛！

眼中闪过一抹狂喜！

"上古遗迹？！"

"而且...是空间系的遗迹！"

"哈哈哈！"

"天助我也！"

...

他站起身...

眼中满是贪婪！

"小子..."

"看来我们之间..."

"还有缘分啊！"

...

话音落下...

他的身形瞬间消失！

向遗迹的方向疾驰而去！

...

而此时的尤溪...

还不知道...

一场更大的危机...

正在等着他！
