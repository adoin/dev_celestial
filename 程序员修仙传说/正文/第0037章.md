# 第37章 妖兽围攻

"吼——！"

巨猿的咆哮声震耳欲聋，整个山谷都在颤抖。

它那三丈高的身躯如同一座小山，每一步踏下都让地面龟裂。

"左边黑豹速度最快，右边蟒蛇有毒，中间巨猿力量最强..."

尤溪的大脑飞速运转，Potato正在实时分析三只妖兽的战斗数据。

"建议战术：先解决速度型，再对付力量型，最后处理毒物。"

"石师姐，你对付黑豹，我来拖住巨猿和蟒蛇！"

"不行！"

石鸾果断拒绝。

"那只巨猿是筑基中期，你对付不了！"

"相信我！"

尤溪转头看着她，眼神前所未有的认真。

"我有办法。"

石鸾看着他的眼睛，心中莫名一颤。

那双眼睛里，没有恐惧，只有冷静和自信。

仿佛...他真的有什么底牌一样。

"...好！"

她咬牙点头。

"但你要是敢死，我就...我就把你鞭尸！"

尤溪："...师姐你这诅咒真别致。"

...

三只妖兽已经扑了上来！

"嗖！"

黑豹化作一道黑影，速度快得肉眼难辨，直取石鸾的咽喉！

"来得好！"

石鸾冷喝一声，手中长剑出鞘，剑光如匹练般斩出！

"锵！"

黑豹的利爪与长剑碰撞，发出金铁交鸣之声。

石鸾被震退数步，但黑豹也被逼退。

另一边——

巨猿和蟒蛇同时攻向尤溪！

"Potato，计算攻击轨迹！"

"计算中..."

"巨猿攻击路径：正面直线，预计0.5秒后到达。"

"蟒蛇攻击路径：侧面缠绕，预计1.2秒后到达。"

"建议：向右闪避0.8米，可躲开双重攻击。"

尤溪身形一闪，精准地向右移动了0.8米。

"轰！"

巨猿的拳头砸在他刚才站立的位置，地面瞬间崩裂！

蟒蛇的尾巴同时扫过，但也被尤溪提前避开。

"什么？"

巨猿猩红的眼睛中闪过一丝惊讶。

这个人类...居然预判了它们的攻击？

"轮到我了！"

尤溪双手结印，体内的五行灵力疯狂涌动。

"五行术法·火球连弹！"

数十个火球从他手中射出，铺天盖地地砸向巨猿！

但这些火球砸在巨猿身上，只是烧掉了几根毛发，根本没造成实质伤害。

"哈哈哈！"

巨猿狂笑。

"区区筑基初期，也敢对本王出手？"

"你的攻击，连给我挠痒痒都不够！"

尤溪脸色微变。

果然，境界差距太大了。

他的攻击对筑基中期的妖兽根本无效！

"Potato，有什么办法？"

"分析中..."

"建议：拖延时间，等待石鸾解决黑豹后来支援。"

"当前石鸾战斗数据：已压制黑豹，预计3分钟内可击杀。"

3分钟？

尤溪苦笑。

他能在巨猿和蟒蛇的联手攻击下撑3分钟吗？

...

战斗持续进行。

尤溪凭借Potato的精准预判，不断闪避着两只妖兽的攻击。

但这样下去不是办法。

他的灵力在不断消耗，而妖兽的体力几乎无穷无尽。

"该死..."

尤溪额头冒出冷汗。

"Potato，还有别的办法吗？"

"检测到石鸾状态异常..."

"警告：石鸾左肩受伤，战斗力下降20%！"

什么？！

尤溪猛地转头看向石鸾的方向。

只见石鸾的左肩被黑豹的利爪抓出了三道血痕，鲜血染红了青色的衣裙。

她的脸色苍白，动作也开始变慢。

"师姐！"

尤溪心中大急。

"别管我！"

石鸾咬牙坚持。

"专心对付你的敌人！"

但她的话音刚落，黑豹再次发动了猛烈的攻击！

"嗖嗖嗖！"

黑豹的速度太快，化作无数道残影，从各个方向攻向石鸾！

石鸾勉强抵挡，但左肩的伤势严重影响了她的动作。

"噗！"

黑豹的利爪再次划过她的手臂，带起一蓬鲜血！

"师姐！"

尤溪目眦欲裂。

他想冲过去帮忙，但巨猿和蟒蛇死死缠住了他。

"想救她？"

巨猿狞笑。

"先过我们这一关！"

"轰！"

巨猿的拳头带着破空之声砸来，尤溪勉强闪避，但还是被拳风扫中，整个人飞了出去！

"噗！"

他口中喷出鲜血，感觉肋骨断了好几根。

"尤溪！"

石鸾看到这一幕，心神大乱。

黑豹抓住这个机会，猛然扑向她！

"嗤！"

利爪穿透了石鸾的腹部！

"呃啊！"

石鸾发出一声惨叫，整个人倒飞出去，重重摔在地上。

"师姐！！"

尤溪的眼睛瞬间红了。

他挣扎着爬起来，不顾一切地冲向石鸾。

"Potato！给我计算最快的救援路线！"

"计算中..."

"路线已生成。"

"但警告：按照当前速度，您将在0.5秒后到达，但石鸾将在0.3秒后遭受致命攻击。"

"来不及！"

尤溪的心沉到了谷底。

黑豹已经再次扑向倒地的石鸾，利爪直奔她的咽喉！

就在这千钧一发之际——

"滚开！！"

一道青色的身影突然从旁边冲出，挡在了石鸾面前！

"噗！"

利爪穿透肉体的声音响起。

但受伤的不是石鸾。

而是突然出现的...顾馨？！

"顾师妹？！"

尤溪和石鸾同时惊呼。

顾馨的胸口被黑豹的利爪贯穿，鲜血狂涌。

但她的双手死死抱住了黑豹，为石鸾争取了宝贵的时间。

"快...快走..."

顾馨艰难地说道。

"我来...拖住它..."

黑豹愤怒地咆哮，疯狂地撕扯着顾馨的身体。

"不——！"

尤溪发出一声怒吼。

体内的某种东西...在这一刻被点燃了。

"Potato！给我把所有算力都用在逃生路线上！"

"收到！"

"全算力 redirected to 路径计算模块！"

尤溪抱起受伤的石鸾，朝着Potato计算出的方向狂奔！

"想跑？"

巨猿和蟒蛇立刻追了上来！

"左侧30度，跳跃！"

"前方5米，翻滚！"

"右侧，急转！"

尤溪按照Potato的指示，在复杂的地形中疯狂穿梭。

巨猿和蟒蛇虽然速度快，但体型太大，在这种地形中反而受到了限制。

"该死的人类！"

巨猿愤怒地咆哮。

"别想逃！"

它猛地一拳砸向地面！

"轰隆隆！"

恐怖的力量传导开来，地面崩裂，石块飞溅！

一块巨石砸中了尤溪的后背！

"噗！"

尤溪再次喷出一口鲜血，但他咬牙坚持，继续奔跑。

"尤溪...放下我...你自己跑..."

石鸾虚弱地说道。

她的伤势太重了，腹部的伤口不断流血，脸色苍白如纸。

"闭嘴！"

尤溪第一次对石鸾发火。

"我说过，不会让你有事的！"

"你要是敢死，我就...我就真的鞭你的尸！"

石鸾愣了一下，随即露出一个虚弱的笑容。

"你...学我..."

"少废话！保存体力！"

尤溪抱着她，拼命奔跑。

基础辅助的提示不断响起：

"前方发现隐蔽洞穴！"

"建议：进入躲避！"

尤溪眼睛一亮，加速冲向那个洞穴。

...

洞穴入口很小，巨猿和蟒蛇根本进不来。

尤溪抱着石鸾钻了进去，在黑暗中摸索前行。

身后传来巨猿愤怒的咆哮声，但声音越来越远...

"安全了...暂时..."

尤溪松了一口气，将石鸾轻轻放下。

"师姐，你怎么样？"

石鸾已经昏迷过去了。

她的腹部伤口很深，鲜血不断涌出，生命气息在快速流逝。

"师姐！师姐！"

尤溪慌乱地呼唤，但石鸾没有任何反应。

"Potato！快想办法！"

"分析中..."

"石鸾伤势严重，失血过多，内脏受损。"

"预计生存时间：不超过2个时辰。"

"建议：寻找疗伤丹药或灵草，否则..."

Potato没有说完，但尤溪已经明白了。

否则，石鸾会死。

"该死！该死！"

尤溪握紧拳头，指甲深深嵌入掌心。

他看着昏迷的石鸾，心中涌起无尽的自责。

都是因为他...

如果不是为了保护他，石鸾不会受这么重的伤...

"我一定要救你..."

"一定！"

他抱起石鸾，朝着洞穴深处走去。

那里，Potato探测到了一股微弱的灵力波动...

也许，有救命的希望。

（本章完）
