# 第73章 石鸾站队

青牛真人的话让尤溪愣住了。

"研究虚空之道？"

"没错。"青牛真人点点头，"本座研究虚空之道已有数百年，但始终无法入门。既然你有空间天赋，不如我们合作，你帮本座研究虚空之道，本座保你平安，如何？"

尤溪心中一动。这是一个机会！

"Potato，分析这个提议。"

"分析中...青牛真人的提议对宿主有利。与其被迫交出传承，不如暂时合作，拖延时间。建议：接受提议，但要争取更多有利条件。"

尤溪沉思片刻，说道："青牛真人，我可以帮你研究虚空之道，但我有几个条件。"

"说。"

"第一，不得伤害碧寒宫任何人。"

"可以。"

"第二，给我自由出入的权利，不得限制我的行动。"

青牛真人眉头一皱："这个..."

"第三，"尤溪继续说道，"如果研究出了成果，我要分享一部分。"

青牛真人沉默了。

这小子，胆子倒是不小，竟然敢跟自己谈条件。

但他也不得不承认，尤溪的条件并不算过分。

"好，本座答应你。"青牛真人点点头，"但你也要答应本座，不得耍花样，否则...

他眼中闪过一抹寒光。

尤溪心中一凛，但表面上依然平静："明白。"

...

碧寒宫，议事大殿。

"三长老，您终于出关了！"玄清子看着石破天，脸上露出欣喜之色。

"大长老，情况我都知道了。"石破天沉声说道，"那青牛真人，老夫来对付。"

"三长老有把握？"

"没有十足的把握，但总要试一试。"石破天说道，"尤溪那小子是为了保护碧寒宫才被抓走的，我们不能不管。"

"说得好！"一个清脆的声音传来。

众人转头望去，只见石鸾大步走进大殿，身后还跟着几名弟子。

"鸾儿，你怎么来了？"石破天问道。

"父亲，我也要去救尤溪！"石鸾坚定地说道。

"胡闹！"石破天怒道，"那是金丹期强者的洞府，你去了只会添乱！"

"我不怕！"石鸾倔强地说道，"尤溪是为了救我才受伤的，我欠他一条命！"

"你..."

"三长老，"玄清子开口说道，"让鸾儿去吧。"

"大长老？"

"尤溪和鸾儿之间的情谊，我们都看在眼里。"玄清子叹了口气，"让他们一起去吧，也许...能起到意想不到的作用。"

石破天沉默片刻，最终点了点头："好吧，但你必须听我的指挥，不得擅自行动！"

"是，父亲！"

...

与此同时，青牛真人的洞府中。

尤溪正在一间石室中研究虚空之道。虽然是合作关系，但青牛真人还是派了几名弟子监视他。

"Potato，分析这个洞府的结构。"

"分析中...洞府位于一座灵山之下，共有三层。第一层是客厅和修炼室，第二层是藏书阁和炼丹房，第三层是禁地，青牛真人平时就在那里修炼。洞府的防御阵法极其强大，以宿主目前的修为，正面突破的可能性为零。"

尤溪点点头。想要逃出去，只能另想办法。

"尤溪，研究的怎么样了？"青牛真人的声音从门外传来。

"有一些进展。"尤溪说道，"但还需要更多的时间。"

青牛真人走进石室，看着尤溪面前的虚空令，眼中闪过一抹炽热。

"这就是虚空令？"

"没错。"尤溪点点头，"只有持有虚空令，才能参悟虚空镜。"

"虚空镜..."青牛真人喃喃自语，"传说中虚空宗的至宝，蕴含着完整的空间法则..."

他突然看向尤溪："你有没有想过，虚空宗为什么会覆灭？"

尤溪一愣："不是因为内乱吗？"

"内乱只是表象。"青牛真人摇摇头，"真正的原因是...虚空宗发现了一处上古遗迹，里面藏着足以改变修仙界格局的秘密。为了争夺那个秘密，各大宗门联手覆灭了虚空宗。"

尤溪心中一震。

原来虚空宗覆灭的真相，竟然是这样？

"那个秘密...是什么？"

"不知道。"青牛真人苦笑，"本座研究了数百年，也只查到这些。但我相信，那个秘密一定与虚空之道有关。"

他看着尤溪，认真地说道："小子，如果你能帮本座解开这个秘密，本座愿意付出任何代价！"

尤溪沉默了。

他突然意识到，青牛真人并不是一个纯粹的恶人。他只是一个对虚空之道极度痴迷的修士，为了追求真理，不惜一切代价。

"青牛真人，我会尽力。"尤溪说道。

...

三天后，碧寒宫的队伍出发了。

玄清子、石破天、石鸾，以及二十名筑基后期的精锐弟子，悄悄向青牛真人的洞府潜去。

"父亲，青牛真人的洞府在哪里？"石鸾问道。

"根据探子的回报，就在前方的青牛山中。"石破天说道，"那里是青牛真人的老巢，防御森严，我们必须小心行事。"

"是！"

队伍悄悄接近青牛山，但在距离山脚还有数里的时候，突然被发现了。

"何人胆敢擅闯青牛山？"

一道威严的声音响彻山林，紧接着，数道身影从山中飞出，拦住了碧寒宫众人的去路。

"是青牛真人的弟子！"玄清子沉声说道，"准备战斗！"

"慢着！"石破天突然开口，"我们此次前来，是为了与青牛真人谈判，不是来打架的。"

他上前一步，朗声说道："碧寒宫三长老石破天，求见青牛真人！"

青牛山的弟子们面面相觑，其中一人说道："等着，我去通报。"

...

洞府中，尤溪正在研究虚空之道，突然听到外面的动静。

"Potato，检测外面的情况。"

"检测中...发现碧寒宫众人，玄清子、石破天、石鸾都在其中。他们正在与青牛真人的弟子对峙。"

尤溪心中一喜。碧寒宫来救他了！

但他随即又担心起来。青牛真人是金丹期强者，碧寒宫的人能救他出去吗？

"尤溪，有人来救你了。"青牛真人突然出现在石室门口，脸上带着似笑非笑的表情。

"青牛真人..."

"放心，本座既然答应过你，就不会伤害他们。"青牛真人说道，"但本座也想看看，碧寒宫为了你，愿意付出多大的代价。"

他大手一挥，石室的禁制消失了。

"走吧，一起去见见你的朋友们。"

...

青牛山脚下，气氛剑拔弩张。

青牛真人带着尤溪出现在众人面前，顿时引起一片骚动。

"尤溪！"石鸾惊喜地叫道，想要冲过去，却被石破天拦住。

"青牛真人，"玄清子沉声说道，"尤溪是我碧寒宫的弟子，请放他回去。"

"放他回去？"青牛真人笑了笑，"玄清子，尤溪是自愿跟本座走的，本座为何要放？"

"你..."

"不过，"青牛真人话锋一转，"如果碧寒宫愿意付出一些代价，本座也不是不能商量。"

"什么代价？"

"虚空镜的参悟权。"青牛真人直接说道，"只要让本座参悟虚空镜三个月，本座就放尤溪回去。"

此言一出，碧寒宫众人脸色大变。

虚空镜是碧寒宫的至宝，岂能轻易示人？

"青牛真人，这个条件..."玄清子犹豫不决。

"大长老，不能答应他！"一名长老急道，"虚空镜事关重大，万一..."

"大长老！"石鸾突然开口，"弟子有话要说！"

众人看向她。

石鸾深吸一口气，朗声说道："尤溪是为了保护碧寒宫才被抓走的，我们不能忘恩负义！虚空镜虽然重要，但比起尤溪的性命，又算得了什么？"

"鸾儿，你..."石破天惊讶地看着女儿。

"父亲，女儿知道自己在说什么。"石鸾坚定地说道，"尤溪是女儿的朋友，也是碧寒宫的恩人。如果没有他，碧寒宫可能已经不在了！"

她转向玄清子："大长老，弟子请求您，答应青牛真人的条件！"

玄清子沉默良久，最终叹了口气："好，老夫答应你。"

"大长老！"

"不必多说！"玄清子摆摆手，"尤溪对碧寒宫有大恩，老夫不能见死不救。"

他看向青牛真人："青牛真人，老夫答应你的条件。但你也要遵守承诺，三个月后，放尤溪回去。"

青牛真人满意地点点头："好，一言为定！"

尤溪看着这一幕，心中涌起一股暖流。

碧寒宫，没有放弃他。

石鸾，为了他不惜与长老会争辩。

这份情谊，他铭记于心。

"师姐，谢谢你..."尤溪低声说道。

石鸾看着尤溪，眼中闪着泪光："傻瓜，我们是朋友啊..."

（本章完）
