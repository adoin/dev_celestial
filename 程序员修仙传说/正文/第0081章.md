# 第81章 修仙盟召令

"修仙盟召令？"

我看着手中的金色卷轴，眉头微皱。

这卷轴是今天早上送来的，上面盖着修仙盟的大印，内容很简单：三日后，修仙盟召开"万仙大会"，邀请各宗门代表参加。

而碧寒宫的代表名单中...赫然有我的名字。

"Potato，分析这份召令。"

"分析中...修仙盟，修仙界最高权力机构，统领各大宗门。万仙大会每十年召开一次，讨论修仙界重大事务。本次大会提前召开，原因不明。"

"提前召开？"

"是的。按照惯例，万仙大会应该在下一年举行。"

我眯起眼睛。

提前召开大会，还邀请我参加...这事情不简单。

"尤溪！"青牛真人的声音从门外传来。

"师父。"

青牛真人走进石室，脸色凝重："你也收到召令了？"

"是的，师父。"

"嗯..."青牛真人沉吟片刻，说道，"这次大会，本座也要参加。你...跟本座一起去。"

"弟子遵命。"

"不过，"青牛真人看着我，眼中闪过一抹深意，"到了修仙盟，你要小心说话。那里高手如云，一句话说错，可能就会惹来杀身之祸。"

"弟子明白。"

...

青牛真人离开后，我立刻开始分析当前的局势。

修仙盟突然召开大会，还邀请我参加...这肯定不是巧合。

"Potato，查询最近修仙界的重大事件。"

"查询中...最近三个月修仙界重大事件：1.血魔宗与正道联盟冲突升级；2.虚空宗传承现世；3.青牛真人收徒；4.魔门活动频繁..."

"虚空宗传承现世..."

我摸着下巴，若有所思。

看来，我得到虚空宗传承的事情，已经传遍整个修仙界了。

这次大会，很可能是冲着我来的。

"宿主，建议提高警戒等级。"

"正有此意。"

...

当天晚上，石鸾突然回来了。

"师姐？你怎么回来了？"

"我不放心你。"石鸾走进石室，一脸担忧，"听说修仙盟要召开大会，还点名要你参加？"

"嗯，我也是刚收到消息。"

"太危险了！"石鸾急道，"修仙盟那种地方，金丹期强者比比皆是，万一有人对你不利..."

"师姐，别担心。"我安慰道，"有青牛真人陪着，应该不会有事的。"

"可是..."

"而且，"我笑了笑，"我现在也不是好惹的。"

石鸾看着我自信的样子，虽然还是有些担心，但也稍微放心了一些。

"那你一定要小心。"

"我知道。对了，师姐，碧寒宫那边情况怎么样？"

"还好。"石鸾说道，"血魔宗最近没什么大动作，宗门正在休整。大长老让我告诉你，如果你遇到什么困难，随时可以回碧寒宫。"

我心中一暖。

碧寒宫，永远是我在这个修仙界的后盾。

"替我谢谢大长老。"

...

第二天，我开始为大会做准备。

"Potato，整理所有关于修仙盟的情报。"

"收到。修仙盟情报整理中..."

很快，一份详细的资料出现在我眼前。

修仙盟，总部位于中州修仙城，是整个修仙界的权力中心。盟主尚琼易，金丹后期修为，号称修仙界第一强者。

"尚琼易..."我念着这个名字，心中隐隐有些不安。

这个名字，给我一种很危险的感觉。

"继续整理。修仙盟的主要势力分布、各派系关系、近期动向..."

"收到。"

...

两天时间转瞬即逝。

出发的日子到了。

青牛真人带着我和几名弟子，乘坐飞舟向中州方向飞去。

飞舟上，青牛真人把我叫到船舱中。

"尤溪，到了修仙盟，你要记住几点。"青牛真人严肃地说道。

"师父请说。"

"第一，不要轻易暴露你的能力。修仙盟鱼龙混杂，如果让他们知道你能预测空间坐标，肯定会有人打你的主意。"

"弟子明白。"

"第二，不要得罪尚琼易。那个人...深不可测，就算是本座，也不愿意与他为敌。"

我心中一凛。

连青牛真人都忌惮的人...看来这个尚琼易，真的不简单。

"第三，"青牛真人顿了顿，"如果有人问起虚空宗传承的事情，你说是从本座这里学的。"

我愣了一下，随即明白了青牛真人的用意。

他是在保护我。

如果我说传承是自己得到的，肯定会引来无数觊觎；但如果说是青牛真人教的，那觊觎的人就要掂量掂量了。

"多谢师父。"

"不必谢本座。"青牛真人摆摆手，"你帮本座救回了道侣，本座自然会护着你。"

...

飞舟飞行了整整一天，终于抵达了中州地界。

远远望去，一座巨大的城市矗立在平原之上，气势恢宏，灵气充沛。

这就是修仙城，整个修仙界最繁华的地方。

"好壮观..."我忍不住感叹。

"这里聚集了修仙界最顶尖的势力。"青牛真人说道，"各大宗门、商会、拍卖行...应有尽有。"

飞舟缓缓降落在城外的专用港口。

我们刚下飞舟，就有一名身穿白袍的修士迎了上来。

"青牛真人，久候了。请随我来。"

"有劳。"

...

修仙城内，比我想象的还要繁华。

街道两旁是各种商铺，贩卖着灵丹、法器、功法...空气中弥漫着淡淡的灵气，让人心旷神怡。

"这里真不错..."我小声嘀咕。

"别东张西望。"青牛真人低声提醒，"保持警惕。"

我连忙收敛心神，跟在青牛真人身后。

很快，我们来到了一座宏伟的建筑前。

修仙盟总部。

建筑高达百丈，通体由白玉建成，散发着淡淡的光芒。门口站着两排守卫，每一个都是筑基后期以上的修为。

"好大的排场..."我暗暗咋舌。

"青牛真人到——！"

门口的守卫高声通报。

我们走进大门，穿过一条长长的走廊，来到了一个巨大的大厅。

大厅中已经聚集了数百人，每一个都是各大宗门的精英。金丹期强者就有十几个，筑基期更是数不胜数。

"好多人..."我扫视了一圈，心中暗暗警惕。

"青牛兄，好久不见。"

一个爽朗的声音传来，我转头看去，只见一个身穿紫袍的中年男子走了过来。

"紫云真人。"青牛真人点点头，"别来无恙。"

"哈哈，托你的福，还不错。"紫云真人笑着，目光落在我身上，"这位就是你新收的弟子？那个得到虚空宗传承的小子？"

我心中一凛。

果然，我的名字已经传开了。

"正是。"青牛真人说道，"尤溪，见过紫云真人。"

"晚辈尤溪，见过紫云真人。"

"嗯，不错。"紫云真人打量了我一番，点点头，"年纪轻轻就有如此修为，未来可期啊。"

"真人过奖了。"

...
与紫云真人寒暄了几句，我们找了个位置坐下。

我仔细观察着周围的人群，试图收集更多的情报。

"Potato，标记大厅内所有金丹期强者。"

"收到。已标记12名金丹期强者。其中金丹后期3名，金丹中期5名，金丹初期4名。"

12名金丹期！

这个阵容，足以横扫任何一个中等宗门了。

"那个坐在主位旁边的人是谁？"我注意到一个身穿黑袍的老者，气息深不可测。

"血魔宗长老，血厉，金丹后期。"

血魔宗？！

我心中一紧。

血魔宗的人竟然也来了？而且就坐在主位旁边？

看来这次大会，真的不简单。

"各位——！"

一个威严的声音响彻大厅，所有人都安静了下来。

我抬头看去，只见一个身穿金色长袍的中年男子从后殿走出，缓步走向主位。

他面容俊朗，气质出尘，一举一动都带着一种与生俱来的威严。

尚琼易！

修仙盟盟主，金丹后期巅峰，号称修仙界第一强者！

"诸位远道而来，辛苦了。"尚琼易在主位上坐下，淡淡地说道，"本次大会提前召开，是因为修仙界发生了一些...重要的事情。"

他的目光扫过全场，最后...落在了我身上。

那一瞬间，我感觉像是被一只无形的大手掐住了喉咙，浑身发冷。

"Potato，警告！检测到极强精神锁定！"

"我知道！"

我强忍着不适，与尚琼易对视。

他的目光平静而深邃，仿佛能看透我的一切。

然后，他微微一笑。

"听说，有人得到了虚空宗的传承？"

全场瞬间安静下来，所有人的目光都集中在我身上。

我的心，沉到了谷底。

（本章完）
