# 第53章 下山准备

清晨的阳光洒进修炼室。

尤溪从入定中醒来，感受着体内澎湃的灵力。

筑基中期巅峰...

距离筑基后期只有一步之遥！

"Potato，宗门大比还有几天？"

"距离宗门大比还有15天。"

"建议：尽快完成下山历练的申请手续。"

...

15天...

时间紧迫！

尤溪起身，整理了一下衣衫，向宗门事务堂走去。

...

事务堂内。

负责历练申请的长老看着尤溪，眼中带着惊讶。

"尤溪？你要申请下山历练？"

"是。"

尤溪恭敬地将申请表递上。

"弟子想趁宗门大比前的这段时间，去凡间历练一番。"

...

长老皱起眉头。

"宗门大比在即，你不留在宗门准备，反而要下山？"

"弟子正是为了大比做准备。"

尤溪不慌不忙地解释。

"弟子修炼的功法需要在红尘中历练，磨砺道心。"

"而且，弟子听闻凡间近期有妖兽作乱，想去斩妖除魔，积累功德。"

...

长老沉吟片刻，点头。

"好吧。"

"你的申请本座批准了。"

"但有一个条件——"

"必须在大比前7天返回宗门。"

"否则，取消参赛资格！"

...

"弟子明白！"

尤溪恭敬行礼。

"多谢长老！"

...

申请通过！

尤溪拿着批准文书，兴冲冲地回到洞府。

"师姐！师姐！"

他推开门，大声喊道。

"申请通过了！我们可以下山了！"

...

石鸾从里屋走出，脸上也带着笑意。

"这么顺利？"

"那当然！"

尤溪得意地晃了晃手中的文书。

"你师弟我现在的面子，可大着呢！"

...

石鸾白了他一眼。

"少臭美了。"

"走吧，去收拾行李。"

...

两人开始准备下山的东西。

丹药、符箓、法器...

一样都不能少！

"Potato，生成下山装备清单！"

"收到！"

"推荐携带："

"1. 回春丹×10（疗伤用）"

"2. 疾行符×5（逃跑用）"

"3. 隐匿符×5（潜行用）"

"4. 爆裂符×3（战斗用）"

"5. 阵盘×1（布置观察节点用）"

"6. 灵石×500（备用）"

"7. 换洗衣物×3套"

"8. 师姐×1（必备）"

...

尤溪："？？？"

"最后那个是什么鬼？！"

"根据分析，师姐的存在可以显著提升宿主的战斗力、生存率和心情愉悦度。"

"因此，列为必备物品。"

...

尤溪忍不住笑出声。

这Potato...

越来越懂他了！

...

两人收拾妥当，正准备出发。

突然——

洞府外传来一个清脆的声音。

"尤溪师兄，你在家吗？"

...

尤溪和石鸾对视一眼。

这个声音...

是顾馨！

...

尤溪开门，看到顾馨站在门外，一身粉色长裙，笑靥如花。

"顾师妹？你怎么来了？"

"听说尤师兄要下山历练，小妹特意来送送。"

顾馨眨着大眼睛，一脸天真。

"顺便...小妹也想下山历练，不知道能不能和师兄同行？"

...

尤溪心中一凛。

这女人...

怎么知道他要下山？

"顾师妹的消息真灵通啊。"

尤溪似笑非笑地看着她。

"呵呵，宗门里的事，小妹略知一二。"

顾馨笑道。

"尤师兄，带上小妹一起吧，多个人多个照应嘛。"

...

石鸾从屋内走出，冷冷地看着顾馨。

"顾师妹，我和尤溪两个人就够了。"

"不需要第三者。"

...

顾馨看到石鸾，眼中闪过一丝嫉妒。

但面上依然笑盈盈的。

"石师姐也在啊..."

"那正好，三个人一起，更安全嘛。"

...

石鸾正要拒绝，尤溪却摆了摆手。

"顾师妹，我们此行凶险，不适合带外人。"

"你还是留在宗门吧。"

说完，他拉着石鸾的手，径直离开。

...

顾馨站在原地，脸上的笑容渐渐消失。

取而代之的，是一抹阴冷。

"尤溪..."

"你以为拒绝我就行了？"

"我顾馨想要的东西，从来没有得不到的！"

...

她取出一枚传音符，低声说了几句。

然后...

嘴角勾起一抹意味深长的笑容。

...

尤溪和石鸾离开宗门，向凡间飞去。

"尤溪，那个顾馨明显有问题。"

石鸾皱眉道。

"我知道。"

尤溪点头。

"她出现的时机太巧合了，很可能是温如玉派来的。"

"那我们..."

"没关系。"

尤溪笑道。

"有Potato在，任何跟踪都无所遁形。"

"正好...我们可以将计就计！"

...

他眼中闪过一抹精光。

温如玉派人来跟踪？

好！

就让他看看，什么叫程序员反侦察！

...

两人飞行了约两个时辰，来到一座繁华的凡人城市。

青云城。

这是方圆千里最大的凡人城市，人口超过百万。

"Potato，扫描城市灵气分布！"

"收到！"

"扫描中..."

"发现异常：城市下方存在微弱的灵气脉络，疑似小型灵脉分支。"

"发现修仙者气息×12，修为从炼气到筑基不等。"

"发现妖兽气息×0，城市内没有妖兽活动迹象。"

...

尤溪满意地点点头。

这座城市...

正是建立第一个观察节点的理想地点！

"走，进城！"

他拉着石鸾的手，向城门走去。

...

但就在这时——

Potato突然发出警报！

"警告！检测到跟踪者！"

"距离：500米"

"修为：筑基中期"

"推测：温如玉派来的杀手！"

...

尤溪眼中闪过一抹冷意。

来得真快！

"师姐，有尾巴。"

他低声说道。

"等进城后，我们分开行动。"

"你引开他，我布置观察节点。"

石鸾点头。

"小心。"

"放心。"

尤溪笑道。

"这种小角色，我还不放在眼里。"

...

两人进入青云城，消失在熙熙攘攘的人群中。

身后...

一个黑影悄然跟随。

但他的脚步，却在城门处停住了。

"奇怪..."

"人呢？"

...

他不知道的是——

尤溪和石鸾，已经通过一条隐秘的小巷，来到了城市的另一边。

"Potato，启动反跟踪程序！"

"收到！"

"生成虚假踪迹...投放中..."

"跟踪者已被引向错误方向！"

...

尤溪嘴角勾起一抹得意的笑容。

想跟踪程序员？

天真！

（本章完）
