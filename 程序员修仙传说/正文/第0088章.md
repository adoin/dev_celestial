# 第88章 大会比试

丹胚凝结成功的第二天，我还躺在床上装死。

不是我想偷懒，而是昨晚那一下真的把我折腾惨了。

经脉受损、灵气枯竭、精神疲惫...

感觉身体被掏空。

"尤溪！尤溪！"

石鸾破门而入，一脸焦急："大事不好了！"

"咳咳..."我被吓得差点从床上滚下来，"师姐，怎么了？"

"修仙盟宣布要举办天才大会！"石鸾说道，"各宗门的天才弟子都要参加比试，你...你也被点名了！"

我："..."

尚琼易这老狐狸，搞什么飞机？

我都说了要闭关突破，他还给我安排这种破事？

"什么时候？"

"三天后。"

"我不参加。"我翻了个身，"跟他说我要闭关，没空。"

"不行！"石鸾急道，"这是强制性的！不参加就视为对修仙盟不敬，会被取消进入虚空秘境的资格！"

我坐起身，眉头紧皱。

这老狐狸...

是在逼我啊。

"Potato，分析尚琼易的意图。"

"分析中...推测：尚琼易想通过比试，观察使用者的真正实力，以及...系统的辅助能力。"

果然。

这老狐狸还是不放心我，想亲自检验一下我的底牌。

"师姐，"我问道，"这次大会有什么规则？"

"筑基期和金丹期分开比试，各自决出前十名。"石鸾说道，"奖品很丰厚，第一名可以获得一枚'聚灵丹'，能大幅提升突破成功率。"

聚灵丹？

我眼睛一亮。

这可是好东西啊！

如果能得到聚灵丹，我凝结金丹的成功率就能大幅提升！

"好！"我一拍大腿，"我参加！"

石鸾愣了一下："你...你不是说要闭关吗？"

"闭关和比试不冲突。"我笑道，"而且，如果能拿到聚灵丹，突破金丹期的把握就更大了。"

"可是...你的伤..."

"小伤，不碍事。"我摆摆手，"Potato，制定恢复方案，两天内恢复到最佳状态。"

"收到。预计恢复时间：36小时。"

"完美！"

...

接下来的两天，我进入了疯狂的恢复模式。

丹药当饭吃，灵石不要钱似的往里砸，再加上Potato的精准指导...

到第三天早上，我的状态已经恢复到了九成。

"尤溪，"石鸾给我整理着衣服，"这次大会各宗门天才云集，你不要太逞强。"

"放心，我有分寸。"我笑道，"打不过就认输，不丢人。"

"你..."石鸾无奈地叹了口气，"算了，我陪你一起去。"

"师姐你也要参加？"

"我是金丹期，跟你不是一个组。"石鸾说道，"但我在场边看着，万一有人想暗算你...

"

她眼中闪过一抹寒光。

我笑了："师姐，你现在的样子，像是要吃人。"

"少贫嘴。"

...

天才大会的场地设在修仙城中央广场。

我们赶到时，广场已经人山人海，各宗门的弟子、长老、甚至宗主都来了。

"好多人啊..."我感叹道。

"那是当然。"旁边一个路人甲说道，"这可是修仙盟举办的盛会，修仙界有头有脸的人物都来了。"

"听说盟主大人也会亲临观战！"

尚琼易要来？

我心中一凛。

这老狐狸，果然是要亲自观察我。

"尤溪！"

一个熟悉的声音从身后传来。

我回头一看，是温如玉。

"温兄？"我惊讶道，"你也来了？"

"当然。"温如玉笑道，"这种盛会，怎么能少了我？"

他说着，凑到我耳边低声道："尤兄，我听说这次大会是盟主大人亲自安排的，目的就是挑选进入虚空秘境的人选。"

"哦？"

"前十名的弟子，据说都有机会获得虚空令。"温如玉眼中闪过一抹精光，"尤兄，你可要加油啊。"

我心中冷笑。

虚空令？

老子早就有了。

不过这话不能说，我只能点点头："多谢温兄提醒，我会尽力的。"

"那就好。"温如玉拍拍我的肩膀，"对了，尤兄，有件事我要提醒你。"

"什么事？"

"这次大会的裁判，是盟主大人亲自指定的。"温如玉压低声音，"据说...盟主大人特别关注你。"

来了。

果然是在针对我。

"我知道了，多谢温兄。"

...

大会正式开始。

首先是抽签分组。

筑基期组一共有128人参加，分成8个小组，每组16人，采用淘汰赛制。

我抽到了第三组，对手名单很快就出来了。

"Potato，分析对手实力。"

"收到。正在收集对手情报..."

"第三组对手分析完成："

"第一轮对手：王猛，散修，筑基中期，擅长近战，力量型。"

"第二轮对手：李小倩，青云宗弟子，筑基后期，擅长水系法术。"

"第三轮对手：周破军，天剑门弟子，筑基后期巅峰，剑修。"

"潜在强敌：赵天霸，修仙盟核心弟子，筑基后期巅峰，据说有金丹期战力。"

我看着分析结果，点点头。

对手都不弱，但...

有Potato在，我有信心一路过关斩将。

"尤溪，第一轮要开始了！"石鸾在台下喊道。

"来了！"

...

第一轮对手王猛是个两米高的大汉，肌肉隆起，看起来像座小山。

"小子，"他咧嘴一笑，露出满嘴黄牙，"识相的就自己认输，省得老子动手。"

我："..."

这台词怎么这么老套？

"Potato，计算最优战斗方案。"

"分析中...对手速度慢、防御高、攻击力强。建议：利用身法游走，寻找破绽一击制胜。"

"收到。"

裁判一声令下："开始！"

王猛大吼一声，像头蛮牛一样向我冲来。

"小子，吃我一拳！"

那拳头足有沙包大，带着呼啸的风声砸向我。

但我早就已经预判到了他的动作。

"左侧三步，下腰。"

基础辅助的提示刚响起，我就已经做出了动作。

砰！

王猛的拳头擦着我的头发掠过，砸在地面上，轰出一个大坑。

"什么？！"王猛一愣。

"右侧两步，出掌。"

我按照Potato的指示，一掌拍在王猛的腰侧。

那里是他的气门所在，被我这一掌拍中，他顿时泄了气。

"嗷——！"

王猛惨叫一声，捂着腰蹲了下去。

"承让。"我抱了抱拳。

裁判宣布："尤溪胜！"

台下一片哗然。

"这小子是谁？一招就解决了王猛？"

"王猛可是筑基中期的高手啊！"

"听说他是青牛真人的弟子，叫尤溪。"

"青牛真人的弟子？难怪..."

我淡定地走下台，仿佛刚才只是做了一件微不足道的小事。

"不错嘛。"石鸾笑着递给我一瓶水，"赢得挺轻松的。"

"小意思。"我笑道，"对手太弱了。"

"别骄傲，后面的对手会越来越强。"

"我知道。"

...

第二轮，我对上了李小倩。

这是个美女，一袭蓝衣，气质清冷，像朵冰山雪莲。

"青牛真人的弟子？"她淡淡地看着我，"希望你能让我尽兴。"

我："..."

又是一个自信过头的。

"开始！"

李小倩双手结印，无数水箭向我射来。

"Potato！"

"水箭轨迹计算完成！躲避路线生成！"

我身形闪动，在水箭的缝隙中穿梭，毫发无伤。

"什么？！"李小倩脸色微变，双手再次结印，"水龙术！"

一条巨大的水龙咆哮着向我冲来。

"有点意思..."我嘴角勾起一抹笑容，"Potato，反击方案！"

"收到！检测到水龙核心弱点：龙首下方三寸！"

"了解！"

我不退反进，迎着水龙冲了上去。

"找死！"李小倩冷笑。

但下一秒，她的笑容就凝固了。

我身形一闪，竟然后发先至，出现在水龙的头顶。

然后——

一掌拍下！

砰！

水龙轰然崩溃，化作漫天水花。

"怎么可能..."李小倩难以置信地看着我。

"水龙术的核心是灵力凝聚点，只要破坏那个点，水龙自然就会崩溃。"我淡淡地说道，"你的法术不错，但破绽太明显了。"

裁判宣布："尤溪胜！"

台下再次哗然。

"这小子也太强了吧？"

"连李小倩都不是他的对手？"

"听说他才筑基后期，怎么感觉比金丹期还厉害？"

我淡定地走下台，心中却在暗暗得意。

有Potato这种外挂，打这种程度的对手，简直不要太轻松。

...

第三轮，我对上了周破军。

天剑门的剑修，筑基后期巅峰，据说剑意已经小成。

"你就是尤溪？"周破军是个冷峻的青年，背负长剑，整个人就像一柄出鞘的利剑。

"正是。"

"很好。"周破军缓缓拔出长剑，"我等你很久了。"

"哦？"

"击败你，我就能证明自己是最强的筑基期！"周破军眼中战意燃烧，"来吧，让我看看你有什么本事！"

"开始！"

周破军动了。

他的剑很快，快到肉眼几乎看不见。

一道剑光闪过，我已经感受到了死亡的气息。

"Potato！"

"警告！剑速超过反应极限！建议：使用空间法术干扰！"

"收到！"

我双手结印，一道空间屏障在身前展开。

叮！

周破军的剑刺在空间屏障上，发出清脆的声响。

"空间法术？"周破军眉头一皱，"有点意思。"

他收剑后退，然后——

"剑意·破空！"

一道璀璨的剑气冲天而起，带着无坚不摧的气势向我斩来。

"Potato！防御方案！"

"建议：多层空间折叠，错位防御！"

"来！"

我疯狂结印，在身前布下三层空间折叠。

剑气斩在空间折叠上，被不断削弱、偏移，最终消散于无形。

"什么？！"周破军震惊了，"连剑意都能挡下？"

"你的剑很强。"我说道，"但空间法术，正好克制实体攻击。"

"是吗？"周破军冷笑，"那就试试这招！"

他再次举剑，但这一次，剑身上燃起了熊熊烈火。

"剑意·焚天！"

火属性剑气？！

我脸色一变。

空间法术最怕的就是属性攻击！

"Potato！紧急方案！"

"建议：闪避！正前方偏左15度，速度提升至极限！"

我拼命催动灵力，向侧方闪去。

轰！

火属性剑气擦着我的身体掠过，轰在防护罩上，发出震耳欲聋的巨响。

"好险..."我额头冒出冷汗。

"还没完！"周破军再次举剑，"剑意·连环！"

无数剑气向我笼罩而来，封锁了我所有的退路。

"Potato！"

"分析中...无法完全闪避！建议：硬抗部分攻击，寻找反击机会！"

"那就来吧！"

我咬牙，催动体内所有灵气，在身前布下一道坚固的空间屏障。

砰砰砰砰！

剑气不断轰击在屏障上，我感觉到自己的灵力在飞速消耗。

这样下去...

必败无疑！

"Potato！我需要更强的防御！"

"建议：激活虚空令，借用空间本源之力！"

"虚空令？在这里？"

"是的！虚空令可以短暂提升使用者的空间法术威力！"

"好！"

我悄悄激活胸口的虚空令。

瞬间，一股浩瀚的空间之力涌入我的体内。

"空间屏障· reinforced！"

我双手结印，一道比之前强大十倍的空间屏障在身前展开。

砰砰砰！

周破军的剑气撞在屏障上，全部被弹开。

"不可能！"周破军难以置信地看着我，"你的灵力怎么突然变强了？！"

"秘密。"我笑了笑，"现在，轮到我了。"

我双手结印，一道空间裂缝在周破军脚下展开。

"什么？！"

周破军大惊失色，想要闪避，但已经来不及了。

他半个身体陷入空间裂缝中，被卡在那里动弹不得。

"你输了。"我淡淡地说道。

周破军挣扎了几下，最终叹了口气："我认输。"

裁判宣布："尤溪胜！晋级四强！"

台下已经炸开了锅。

"这也太强了吧？！"

"连周破军都不是他的对手？"

"这小子到底是什么来头？"

"空间法术...难道他是虚空宗的传人？"

我走下台，石鸾立刻迎了上来。

"没事吧？"她关切地问道，"刚才那几下看起来很惊险。"

"没事。"我摇摇头，"就是消耗有点大。"

"给你。"石鸾递给我一瓶丹药，"快回复一下，四强赛马上就开始了。"

"多谢师姐。"

我吞下丹药，运转功法恢复灵力。

...

四强赛的对阵表很快就出来了。

我对上了一个叫孙无敌的散修，据说实力深不可测。

而另一场，是赵天霸对阵一个名不见经传的小宗门弟子。

"Potato，分析孙无敌的情报。"

"收到。孙无敌，散修，筑基后期巅峰，战斗记录：无。"

"无？"

"是的。此人之前的比赛全部是一招制胜，没有任何人能看到他的底牌。"

"有点意思..."我皱起眉头。

"使用者，根据数据分析，孙无敌有73%的概率隐藏了真实修为。"

"你是说...他可能是金丹期？"

"很有可能。"

"如果是金丹期..."我摸了摸下巴，"那就有点麻烦了。"

不过...

就算他是金丹期，我也不怕。

有Potato在，有虚空令在...

我还真不信会输！

...

四强赛开始。

我先上场，对手孙无敌是个看起来很普通的青年，穿着一身灰袍，没有任何特别的气质。

但当他走上台时，我感受到了一股巨大的压力。

"尤溪？"他淡淡地看着我，"我听说过你。"

"彼此彼此。"

"你的空间法术很有意思。"孙无敌说道，"可惜...

"

他顿了顿，眼中闪过一抹金光："在绝对的实力面前，技巧毫无意义。"

话音刚落，一股恐怖的气势从他身上爆发出来。

金丹期！

真的是金丹期！

台下一片哗然。

"金丹期？！他不是筑基期吗？"

"隐藏修为！这是作弊！"

"裁判！这不公平！"

裁判面无表情地宣布："规则允许隐藏修为，比赛继续。"

我苦笑。

果然...

尚琼易这个老狐狸，故意安排了一个金丹期来试探我。

"Potato，对手是金丹期，胜率多少？"

"分析中...胜率：12%。"

"这么低？"

"孙无敌的修为是金丹初期，但根基很稳固，战斗经验丰富。以使用者目前的实力，正面硬拼几乎没有胜算。"

"那怎么办？"

"建议：利用空间法术周旋，寻找破绽。同时...准备使用虚空令的隐藏功能。"

"隐藏功能？"

"虚空令除了可以撕裂空间，还能短暂扭曲周围的空间规则。如果使用得当，可以干扰金丹期修士的灵力运转。"

"好东西啊！怎么不早说？！"

"该功能消耗极大，使用后会进入24小时的冷却期。"

"不管了，先用了再说！"

我深吸一口气，看向孙无敌："金丹期又如何？来吧！"

"有胆量。"孙无敌点点头，"那我就成全你。"

他身形一闪，瞬间出现在我面前。

好快！

比周破军快了至少三倍！

"Potato！"

"警告！无法预判！自主防御！"

我本能地催动空间屏障。

砰！

孙无敌一掌拍在屏障上，屏障瞬间出现裂痕。

"有点意思。"孙无敌收回手掌，"能挡住我一击，你比那些废物强多了。"

"少废话！"我咬牙，"再来！"

"如你所愿。"

孙无敌再次出手，这一次，他的攻击更加凌厉。

我拼命催动空间法术防御，但金丹期和筑基期的差距太大了。

每一次碰撞，我都感觉自己的经脉在震颤。

再这样下去...

必败无疑！

"Potato！准备虚空令隐藏功能！"

"收到！正在充能...10%...20%..."

"快点啊！"

孙无敌的攻击越来越猛烈，我的空间屏障已经布满裂痕。

"50%...70%...90%..."

"100%！充能完成！"

"就是现在！"

我猛地激活虚空令的隐藏功能。

嗡——

一股奇异的空间波动以我为中心扩散开来。

孙无敌的动作突然一滞。

"什么？！"他脸色微变，"我的灵力..."

"被干扰了吧？"我冷笑道，"这就是空间规则的威力！"

"空间规则？"孙无敌瞳孔一缩，"你怎么可能掌握..."

"比赛还没结束呢！"

我趁着孙无敌被干扰的机会，疯狂发动攻击。

空间裂缝、空间折叠、空间震荡...

各种空间法术不要钱似的往他身上招呼。

孙无敌被打得连连后退，一时间竟落入了下风。

"好！"台下有人喝彩。

"这小子居然能压制金丹期？！"

"太厉害了！"

但好景不长。

孙无敌很快就适应了空间规则的干扰，开始反击。

"小子，你激怒我了。"

他双眼变得赤红，周身灵力疯狂涌动。

"金丹神通·碎星掌！"

一只巨大的灵力手掌从天而降，带着毁天灭地的气势向我拍来。

"Potato！"

"警告！攻击威力超出防御极限！建议：闪避！"

"往哪闪？！"

"正下方！利用空间折叠下沉！"

"来！"

我双手结印，在身下打开一道空间裂缝，整个人瞬间下沉。

轰！

巨大的灵力手掌拍在地面上，整个擂台都在颤抖。

而我...

从空间裂缝的另一端钻了出来，出现在孙无敌身后。

"什么？！"孙无敌大惊。

"结束了！"

我一掌拍在他的后心。

砰！

孙无敌踉跄了几步，但很快就稳住了身形。

"小子..."他转过身，眼中满是怒火，"你很好。"

"但是..."

他再次举起手掌，灵力比之前更加狂暴。

"金丹神通·碎星掌·二连！"

两只灵力手掌同时出现，封锁了我所有的退路。

"Potato！"

"警告！无法闪避！建议：硬抗！"

"硬抗个屁啊！会死的！"

"那...认输？"

"不认！"

我咬牙，将体内所有灵力都注入虚空令。

"空间规则·极限干扰！"

嗡——

前所未有的空间波动爆发开来。

那两只灵力手掌在空间波动中颤抖、扭曲，最终...

消散了！

"什么？！"孙无敌难以置信地看着我，"这不可能！"

"没有什么不可能！"我喘着粗气，"再接我一招！"

我拼尽最后一丝力气，在孙无敌脚下打开一道空间裂缝。

"不好！"

孙无敌想要闪避，但已经来不及了。

他的身体陷入空间裂缝中，被卡在那里动弹不得。

"你..."他愤怒地看着我。

"承让。"我露出一个虚弱的笑容。

裁判宣布："尤溪胜！晋级决赛！"

台下爆发出雷鸣般的掌声。

"太精彩了！"

"筑基期击败金丹期！这是奇迹啊！"

"这小子绝对是这次大会最大的黑马！"

我倒在地上，大口喘着气。

赢了...

虽然赢得极其艰难，但...

我赢了！

石鸾冲上台，把我扶起来："你没事吧？"

"没事...就是有点虚..."我苦笑道，"扶我下去，我要好好休息..."

"决赛怎么办？"

"决赛？"我一愣，"不是已经打完了吗？"

"还有一场决赛啊！"石鸾无奈道，"你对赵天霸！"

"卧槽！"

我差点晕过去。

还有一场？！

就我现在这状态...

怎么打？！
