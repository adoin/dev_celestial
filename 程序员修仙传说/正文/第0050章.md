# 第50章 震惊宗门

"听说了吗？昨天有人在宗门外渡劫！"

"渡劫？谁啊？"

"不知道，但那个动静，简直跟金丹雷劫一样恐怖！"

"不会是有长老突破金丹期了吧？"

碧寒宫内外，到处都在议论昨天的雷劫。

那五道雷劫的威势，整个宗门都看到了。

尤其是最后一道五彩雷劫...

简直像是末日降临！

...

议事大殿内。

玄清子大长老坐在主位上，脸色凝重。

"查清楚了吗？"

"是谁在渡劫？"

下方，一名执法长老躬身回答：

"回大长老，根据现场勘察，渡劫地点正是尤溪三天前离开的方向。"

"而且..."

"我们在现场发现了这个。"

他取出一枚残破的玉佩，上面还残留着焦黑的痕迹。

玄清子接过玉佩，仔细端详。

"这是...尤溪的身份玉牌？"

"正是。"

...

全场哗然！

"尤溪？！"

"难道昨天的雷劫，是他引来的？！"

"不可能吧！筑基期怎么会有雷劫？！"

"但那个动静...确实不像普通的筑基突破..."

...

温如玉站在人群中，脸色铁青。

尤溪...

又是尤溪！

那个杂灵根的废物，凭什么引来雷劫？！

"大长老，弟子认为此事必有蹊跷！"

温如玉再次站出来。

"尤溪一个五行杂灵根的废物，怎么可能引来雷劫？"

"就算是天灵根的天才，筑基期也不会有天劫！"

"弟子怀疑，他一定是修炼了什么邪术，才会引起天地异象！"

...

其他长老也纷纷点头。

确实...

筑基期引来雷劫，闻所未闻！

除非...

他真的修炼了某种禁忌之术！

"大长老，如果尤溪真的修炼邪术，必须严惩！"

"没错，这种邪魔外道，绝不能留在宗门！"

"废除修为，逐出宗门！"

...

玄清子眉头紧锁。

他也觉得此事太过诡异。

但...

没有确凿证据，不能轻易定罪。

"等尤溪回来，本座亲自审问。"

"如果他真的修炼邪术..."

玄清子眼中闪过一抹寒光。

"本座绝不姑息！"

...

就在这时——

大殿外传来一阵脚步声。

众人回头望去...

只见一个浑身焦黑的青年，缓缓走进大殿。

他的衣服破破烂烂，头发像个爆炸头，脸上还有几道烧伤痕迹。

但...

他的眼睛，明亮如星！

他的气息...

筑基中期巅峰！！

...

"弟子尤溪，拜见大长老。"

尤溪单膝跪地，恭敬行礼。

...

全场死寂！

所有人都瞪大了眼睛，看着这个"乞丐"一般的青年。

筑基中期巅峰？！

三天前，他还是筑基初期巅峰！

三天后...

筑基中期巅峰？！

这他妈是什么鬼速度？！

...

玄清子也震惊了。

他活了上百年，从未见过如此恐怖的修炼速度！

"尤溪...你...你真的突破了？"

"回大长老，弟子幸不辱命。"

尤溪抬起头，眼中闪烁着自信的光芒。

"三天前，大长老给弟子三天时间自证清白。"

"弟子不敢懈怠，拼死突破，终于在时限内突破至筑基中期！"

"并且..."

他站起身，一股强大的气息从他身上爆发而出！

"引动了天劫！"

"成功渡过五行雷劫！"

...

"五行雷劫？！"

玄清子猛地站起身！

"你是说...你引来的是五行雷劫？！"

"正是。"

尤溪点头。

"弟子在突破时，五行灵根同时进阶，引起了天地异象。"

"雷劫威力虽然强大，但弟子侥幸渡过。"

...

五行雷劫！！

那可是传说中的天劫！

只有五行灵根的修士，在五行合一的时候，才会引来的特殊雷劫！

这种雷劫...

万中无一！

据说，渡过五行雷劫的修士...

未来必成元婴！

甚至...

化神有望！

...

玄清子看着尤溪，眼中满是复杂。

五行杂灵根...

原本是最废的灵根！

但若是五行合一...

那就是最强的灵根！

"尤溪，你可愿让本座检查一下你的身体？"

"确认你确实没有修炼邪术？"

...

尤溪坦然伸出手。

"请大长老检查。"

玄清子握住尤溪的手腕，一道温和的神识探入他体内。

片刻后...

玄清子收回神识，脸上露出惊叹之色！

"好！好！好！"

他连说三个好字。

"尤溪，你的灵力纯正浑厚，毫无邪术痕迹！"

"你的修为，确实是靠正当修炼得来的！"

...

全场再次哗然！

尤溪...

真的靠正当修炼，在三天内突破了？！

而且引来传说中的五行雷劫？！

这...

这简直是逆天！

...

温如玉脸色铁青。

该死！

这小子竟然真的突破了！

而且还引来五行雷劫！

这不可能！

"大长老，弟子不服！"

温如玉再次站出来。

"就算尤溪没有修炼邪术，他的修炼速度也太不正常了！"

"弟子怀疑，他一定是用了什么禁药或者秘法！"

"否则，怎么可能三天突破？！"

...

尤溪冷笑一声。

"温师兄此言差矣。"

"我的修炼速度，确实比常人快。"

"但这并不是因为我用了什么禁药或者秘法。"

"而是因为..."

他看向温如玉，嘴角勾起一抹嘲讽的弧度。

"我比你有天赋！"

...

"你说什么？！"

温如玉大怒！

他可是单灵根的天才！

尤溪一个五行杂灵根的废物，竟然敢说他比自己有天赋？！

"怎么？"

尤溪挑眉。

"温师兄不信？"

"不如...我们来比划比划？"

"让所有人看看，到底谁更有天赋！"

...

温如玉眼中闪过一抹杀意。

正合他意！

他正愁没机会在大比之前解决尤溪！

现在尤溪主动挑衅...

那就别怪他不客气了！

"好！"

"我就让你知道，什么叫真正的天才！"

...

玄清子皱眉。

"这里是议事大殿，岂容你们胡闹？"

"大长老。"

尤溪拱手。

"弟子只是想证明自己的清白。"

"如果弟子能击败温师兄，就能证明弟子的实力是真实的，不是吗？"

玄清子沉默片刻，点头。

"好。"

"本座准许你们比试一场。"

"但点到为止，不可伤人性命。"

...

众人退出大殿，来到殿前的广场上。

尤溪和温如玉相对而立，气氛剑拔弩张。

"尤溪，我会让你知道，挑衅我的下场！"

温如玉冷冷说道。

"是吗？"

尤溪笑了。

"那我们就拭目以待吧。"

...

"开始！"

随着玄清子一声令下，战斗瞬间爆发！

温如玉率先出手！

"青木剑诀·万剑归宗！"

刹那间——

无数道青色剑气从他体内爆发，如同狂风暴雨般向尤溪袭来！

这一招...

是温如玉的成名绝技！

曾经，他用这一招击败过筑基后期的强者！

...

面对漫天剑气，尤溪却一动不动。

"Potato，分析剑气轨迹。"

"收到！"

"计算完成！"

"共有剑气108道，封锁了使用者所有闪避路线。"

"但其中有三道剑气是虚招，真正的杀招只有九道！"

"建议：硬抗虚招，闪避杀招！"

...

尤溪眼中精光一闪！

在剑气即将临身的瞬间——

他动了！

身形如同鬼魅，在剑气的缝隙中穿梭！

"噗噗噗！"

几道剑气击中他的身体，但都是无关紧要的部位！

而那九道真正的杀招...

全部落空！

...

"什么？！"

温如玉瞪大了眼睛！

他的万剑归宗...

竟然被躲开了？！

这怎么可能？！

"轮到我了！"

尤溪冷笑一声，身形瞬间消失！

"空间折叠·瞬身！"

刹那间——

他出现在温如玉身后！

"五行拳·金破！"

金色的拳影，狠狠砸向温如玉的后心！

...

"该死！"

温如玉仓促转身，双手交叉格挡！

"轰！"

巨大的力量将他震退数步！

他的双臂都在颤抖！

"好强的力量！"

"这小子...什么时候变得这么强了？！"

...

尤溪不给温如玉喘息的机会！

"五行拳·木缚！"

"五行拳·水淹！"

"五行拳·火焚！"

"五行拳·土崩！"

...

金、木、水、火、土...

五种属性的攻击连绵不绝，如同潮水般向温如玉涌去！

温如玉节节败退，狼狈不堪！

"不可能！"

"我怎么会输给你这个废物？！"

他怒吼着，施展出最强的绝招！

"青木剑诀·剑斩山河！"

一道巨大的青色剑气，从天而降！

这一剑...

凝聚了他全部的灵力！

威力之大，足以斩断山河！

...

面对这惊天动地的一剑，尤溪却笑了。

"就这？"

"Potato，计算这一剑的轨迹！"

"计算完成！"

"攻击角度：45度"

"攻击速度：500米/秒"

"威力评估：相当于筑基后期全力一击"

"闪避方案：向左偏移1.5米，然后反击！"

...

尤溪按照Potato的指示，身形微微一侧...

那道惊天剑气，从他身侧掠过，劈在地面上，炸出一个巨大的深坑！

而尤溪...

毫发无伤！

"结束了。"

他轻声说道。

"空间禁锢！"

五行灵力爆发，温如玉的身体瞬间被定在原地！

然后——

尤溪的拳头，停在了温如玉的面前。

距离他的鼻子，只有一寸。

"温师兄，承让了。"

...

全场死寂！

所有人都瞪大了眼睛，看着这一幕。

温如玉...

败了？！

碧寒宫的大师兄，单灵根的天才...

竟然败给了一个五行杂灵根的"废物"？！

...

片刻后...

人群中爆发出震天的欢呼声！

"赢了！尤溪赢了！"

"天啊！他竟然真的击败了温师兄！"

"这是什么妖孽？！杂灵根竟然能击败单灵根的天才？！"

"太燃了！太燃了！"

...

玄清子看着尤溪，眼中满是赞赏。

"好！好一个尤溪！"

"从今天起，你正式晋升为碧寒宫核心弟子！"

"享有筑基弟子的最高待遇！"

...

尤溪收起拳头，向玄清子行礼。

"多谢大长老！"

然后...

他看向温如玉，嘴角勾起一抹冷笑。

"温师兄，记住了。"

"在修仙界..."

"灵根决定不了命运！"

"努力，才是王道！"

...

温如玉脸色铁青，拳头握得咯咯作响。

但他...

无话可说！

他输了！

当着全宗门的面，输给了一个他曾经看不起的废物！

这种屈辱...

比杀了他还难受！

"尤溪..."

他在心中嘶吼。

"宗门大比...我一定让你死无葬身之地！"

...

尤溪没有理会温如玉的怨毒目光。

他转过身，看向人群中的石鸾。

石鸾正看着他，眼中满是骄傲和喜悦。

尤溪向她眨了眨眼，嘴角扬起一抹得意的笑容。

师姐，看到了吗？

我做到了！

...

夕阳西下。

尤溪的身影被拉得很长。

从今天开始...

他不再是那个任人欺凌的杂灵根废物！

他是...

尤溪！

程序员修仙者！

一个注定要在修仙界书写传奇的男人！

（本章完）
