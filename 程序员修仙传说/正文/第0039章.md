# 第39章 洞府奇遇

尤溪的手在颤抖。

他死死盯着玉简中的内容，仿佛看到了什么不可思议的东西。

"空间坐标计算公式..."

"五行定位法..."

"维度折叠理论..."

这些名词，每一个都让他心跳加速！

因为玉简中的内容，完全印证了他之前的猜想——五行术法，本质上是空间系的一种表现形式！

"Potato！快！把这些内容全部扫描储存！"

"收到！"

"开始扫描《空间术法详解》..."

"扫描进度：10%...25%...50%..."

尤溪激动地来回踱步。

有了这些理论支持，他的智能体1.0就可以进入最后的完善阶段了！

"75%...90%...100%！"

"扫描完成！"

"正在整合分析..."

"分析结果："

"该玉简记载了一套完整的空间理论体系。"

"核心观点：五行相生相克，本质是空间维度的不同表现形式。"

"金：空间压缩"

"木：空间生长"

"水：空间流动"

"火：空间爆裂"

"土：空间稳固"

"通过五行术法的组合，可以实现对空间的操控。"

尤溪狂喜！

这简直就是为他量身定做的理论！

他是五行杂灵根，可以同时操控五种属性的灵力。

如果按照这个理论...

他就能成为真正的空间系修士！

"Potato，根据这个理论，能完善智能体1.0的空间模块吗？"

"分析中..."

"可以！"

"该理论提供了完整的空间坐标计算公式，可以大幅提升空间感知的精度。"

"预计完善后，空间感知范围可提升300%，精度提升500%！"

"智能体1.0完成度：可提升至85%！"

85%！

尤溪激动得差点跳起来。

距离完全体，只差15%了！

...

就在他沉浸在喜悦中时，软榻上传来了一声轻吟。

"嗯..."

尤溪连忙转头，发现石鸾醒了！

"师姐！你醒了！"

他冲到软榻边，紧张地看着她。

石鸾缓缓睁开眼睛，美眸中还有些迷茫。

"我...这是在哪？"

她的声音虚弱，但比昏迷前好了很多。

"我们在一个上古修士的洞府里。"

尤溪解释道。

"你受了重伤，我找到了这里的丹药，救了你。"

石鸾低头看了看自己的腹部，发现伤口已经完全愈合，只留下一道淡淡的疤痕。

"这..."

她惊讶地瞪大了眼睛。

"我的伤..."

"是九转还魂丹，四品丹药。"

尤溪笑道。

"你现在的身体应该没什么大碍了，就是还有些虚弱，需要静养。"

石鸾沉默了一会，突然问道：

"我昏迷了多久？"

"三天三夜。"

尤溪如实回答。

"三天三夜？"

石鸾愣住了。

她看向尤溪，发现他的脸色苍白，眼睛里布满了血丝，整个人憔悴不堪。

显然，这三天三夜他一直在守护她，没有合眼。

"你...一直守着我？"

她的声音有些颤抖。

"当然。"

尤溪理所当然地点头。

"我说了不会让你有事的，就得说到做到。"

"而且..."

他挠挠头，有些不好意思。

"你是因为我才受伤的，我要是不管你，那还是人吗？"

石鸾看着他，眼眶突然红了。

"笨蛋..."

她的眼泪夺眶而出，顺着脸颊滑落。

"你才是笨蛋..."

"明明自己也很虚弱，还要硬撑着..."

"要是你也倒下了怎么办..."

尤溪愣住了。

他没想到石鸾会哭。

这个平日里傲娇强势的师姐，此刻却像个受了委屈的小姑娘，哭得梨花带雨。

"师姐...别哭了..."

他手足无措，不知道该怎么安慰。

"我这不是没事吗..."

"你还说！"

石鸾抽泣着，突然扑进了他的怀里。

"以后不许再做这么危险的事了！"

"听到没有！"

尤溪整个人都僵住了。

他能感受到石鸾柔软的身体，能闻到她发间的清香，能听到她急促的心跳...

这...这是什么情况？

"师姐..."

"闭嘴！"

石鸾把脸埋在他怀里，声音闷闷的。

"让我抱一会..."

"就一会..."

尤溪不敢动了。

他僵硬地站在那里，任由石鸾抱着。

良久。

石鸾终于松开了他，脸上还带着泪痕，但已经恢复了平静。

"咳..."

她轻咳一声，别过脸去，耳尖微微泛红。

"刚才...刚才什么都没发生，听见没有？"

尤溪："...听见了。"

"哼。"

石鸾瞪了他一眼。

"还有，不许跟别人说！"

"尤其是那个顾馨！"

尤溪苦笑。

"顾师妹她..."

他顿住了。

顾馨为了保护石鸾，被黑豹贯穿了胸口...

现在生死未卜。

"顾师妹怎么了？"

石鸾察觉到他的异样。

"她...为了救你，挡住了黑豹的攻击。"

尤溪低声说道。

"她...可能已经..."

石鸾沉默了。

她想起了昏迷前的那一幕——顾馨突然冲出来，用身体挡住了黑豹的利爪...

"她是个好人..."

石鸾轻声说道。

"虽然我们之前有过节，但...她救了我的命。"

"等出去后，我们要找到她的尸体，好好安葬。"

尤溪点点头。

...

两人沉默了一会，尤溪转移话题：

"师姐，你看看这个。"

他把《空间术法详解》的玉简递给石鸾。

"这是什么？"

石鸾接过玉简，神识探入其中。

片刻后，她震惊地抬起头。

"这...这是..."

"空间术法的完整理论！"

尤溪激动地说道。

"玉简里记载，五行术法的极致是空间！"

"金木水火土，分别对应空间的不同属性！"

"这完全印证了我之前的猜测！"

石鸾看着他兴奋的样子，眼中闪过一丝复杂的情绪。

"尤溪..."

"嗯？"

"你到底是什么人？"

她认真地问道。

"为什么你知道这么多奇怪的东西？"

"为什么你能破解上古禁制？"

"为什么你...总是能做出让人意想不到的事情？"

尤溪愣住了。

他看着石鸾认真的眼神，心中涌起一股冲动。

要不要告诉她Potato的存在？

这个秘密，他藏在心里太久了...

"我..."

他刚要开口，Potato突然发出警报：

"警告！"

"检测到洞府禁制被触发！"

"有人正在靠近！"

尤溪脸色一变。

"有人来了！"

...

洞府外，传来一阵脚步声。

"奇怪，这里的禁制怎么被人破解了？"

"难道有人先进去了？"

"搜！不能让任何人抢先！"

是人类的说话声！

而且不止一个人！

尤溪和石鸾对视一眼，同时做出了决定——

隐藏起来！

"Potato，有没有隐藏的地方？"

"高台后方有一个暗格，可以暂时躲避。"

两人迅速躲进暗格，屏住呼吸。

...

片刻后，几道身影进入了洞府。

为首的是一个中年男子，筑基后期修为，穿着碧寒宫的服饰。

但尤溪从未见过这个人。

"师兄，这里确实有人来过。"

一个年轻弟子说道。

"看地上的痕迹，应该是刚走不久。"

中年男子冷哼一声。

"不管是谁，敢抢我们的机缘，都得死！"

"搜！把所有值钱的东西都带走！"

"还有，找到那个抢先一步的人，格杀勿论！"

"是！"

几个弟子开始搜刮洞府中的物品。

尤溪躲在暗格中，心中怒火升腾。

这些人...

是冲着洞府的传承来的！

而且听他们的语气，似乎专门在追杀抢先进入的人...

"Potato，能识别他们的身份吗？"

"从服饰判断，是碧寒宫弟子。"

"但从灵力波动分析，与壁寒宫正统功法有差异。"

"推测：可能是壁寒宫的附庸势力，或者...假冒的。"

尤溪眯起眼睛。

不管是谁，这些人明显来者不善。

他和石鸾现在的状态都不好，硬拼肯定不行。

只能先躲着，等待机会。

...

外面的搜刮持续了约半个时辰。

那些人把洞府中的丹药、灵石、法器洗劫一空，连高台上五行道人的遗骸都没放过。

"师兄，这具枯骨..."

"带走！上古修士的骨头也是炼器材料！"

尤溪握紧了拳头，指甲深深嵌入掌心。

这些混蛋...

连死者的遗骸都不放过！

但他不能冲动。

现在出去，只有死路一条。

...

终于，那些人搜刮完毕，离开了洞府。

"师兄，没有找那个抢先进入的人..."

"哼，跑得了和尚跑不了庙。"

"秘境就这么大，迟早能找到他！"

"走，去向少主汇报！"

脚步声渐渐远去。

...

暗格中，尤溪和石鸾松了口气。

"少主？"

石鸾皱眉。

"他们说的少主...难道是..."

"温如玉。"

尤溪冷冷地说道。

"除了他，没人会派人来追杀我。"

"刚才那些人说'向少主汇报'，肯定是温如玉的人。"

石鸾脸色凝重。

"看来...他早就知道寂云谷里有上古传承。"

"派你来这里，就是想借刀杀人，顺便让你当探路石。"

尤溪点头。

"没错。"

"但他没想到，我没死，还得到了传承。"

"现在，他派人来追杀我，想夺取我得到的东西。"

他站起身，眼中闪过一丝寒光。

"温如玉..."

"既然你想玩，那我就陪你玩到底！"

Potato的声音在脑海中响起：

"检测到宿主情绪波动。"

"建议：冷静分析，制定对策。"

"当前首要任务：安全离开秘境，同时躲避追杀。"

"智能体1.0完善进度：可提升至85%，需要时间：约6个时辰。"

尤溪深吸一口气，冷静下来。

"师姐，我们在这里休整一下。"

"我需要时间消化玉简中的内容，提升实力。"

"等准备好了，再出去对付那些追杀者！"

石鸾点点头。

"好，我帮你护法。"

两人对视一眼，眼中都燃起了斗志。

...

就在此时，洞府外突然传来一声震天的咆哮！

"吼——！！"

那声音，比之前遇到的巨猿还要恐怖十倍！

"警告！检测到极强灵压！"

"等级判断：金丹期！"

"警告！金丹期妖兽正在接近！"

尤溪和石鸾同时变色。

金丹期妖兽？！

这寂云谷里，竟然还有这种恐怖的存在！

（本章完）
