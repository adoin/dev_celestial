# 写作规范

## 章节命名规则

### 基本原则
章节名必须是**本章内的爆点台词**，而非总结性标题。

### 要求
1. **必须是台词/想法**：从本章实际内容中提取的一句话
2. **有冲突/转折/悬念**：这句话应该让读者产生"什么？！"的反应
3. **不能是总结性标题**：如"父亲的遗愿""壁寒宫的清晨"这种描述性标题不合格

### 示例对比

❌ **错误示例**（总结性标题）：
- 第3章 父亲的遗愿
- 第4章 壁寒宫的清晨
- 第1章 这里是哪里？

✅ **正确示例**（爆点台词）：
- 第2章 "尤师兄，您昏睡三天了"
- 第3章 "你父亲虚空乱流断后，身死道消了"
- 第5章 "我的...小废物。"

### 操作流程
1. 写完章节后，找出本章最有冲击力的台词
2. 将台词记录在本文档的"章节名清单"中
3. 修改正文的章节名为该台词
4. Git提交时注明章节名修改

## 时间线一致性规则

### 严禁出现的内容
- ❌ "他在第三章尝试..."
- ❌ "第三章里那个想法..."
- ❌ 任何形式的"第X章"引用

### 正确的时间表达
- ✅ "昨天尝试引气时..."
- ✅ "刚才那个想法..."
- ✅ "今天早上..."
- ✅ "不久前..."

### 原因
小说是故事内时间线，不是写作结构的章节编号。"第三章"是元叙事（meta），会打破读者沉浸感。

---

## 剧情连贯性规则

### 不得编造未发生的情节
- ❌ "密封陶罐实验...他在第三章试过" ← 第三章根本没有这个实验
- ❌ 引用不存在的对话或事件

### 正确做法
1. 写作前检查前文是否真的有该剧情
2. 使用Grep工具搜索关键词确认
3. 不确定时重新描述，而非引用

---

## 设定一致性规则

### 修仙世界特殊性
1. **战死方式**：修士战死是"身死道消"（瞬间），不是"卧病在床"（凡人）
2. **境界感知**：修士一探便知他人灵根，无需"测灵石"等道具公开展示
3. **境界差距**：筑基修士无法引开太古魔修（元婴级），境界设定要合理
4. **时间单位**：古代背景，禁用现代单位
   - ❌ 秒、分钟、小时（现代）
   - ✅ 一刻、一炷香、一盏茶、时辰、天、月、年、甲子（古代）
   - 示例："外界三个时辰，秘境内半月之久"

### 人物关系逻辑
1. **同宗门认识**：在同一宗门生活多年的弟子，不需要"自我介绍"
   - ❌ 内门大师兄对内门弟子说"忘了自我介绍，我是XXX"
   - ✅ 直接称呼对方名字/师弟，彼此都知道对方身份
2. **穿越者特殊处理**：主角是穿越者不认识他人时，用"记忆碎片"补充认知
   - 不要让对方"自我介绍"
   - 通过外貌特征、记忆碎片、他人反应让主角认出对方

### 避免的老套路
- ❌ 父亲临终遗言（武侠剧）
- ❌ 公开测灵根仪式（羞辱工具）
- ❌ 弱不禁风的内门弟子（不合理）

## 对话精简原则

### 一般规则
1. 对话不超过3轮（你来我往3次）
2. 背景信息用叙述，不要靠对话堆砌
3. 删除幼稚/情绪化台词

### 示例改进

❌ **冗长版**（54行对话）：
```
"师父...我爹真的回不来了吗？"
"他为什么要去那里？他明明说好..."
老者沉默，缓缓开口："半月前，宗门在苍云峡..."
[...48行详细描述...]
"好孩子..."
```

✅ **精简版**（10行对话）：
```
"你父亲虚空乱流断后，身死道消了。宗门欠他的。"
老者递出玉牌："从今天起，你是内门弟子。"
"可是...我是杂灵根..."
"我知道。宗门会按内门弟子待遇供养你。"
"我会好好修炼的。"
```

## 行文密度规则

### 核心禁令：禁止用换行代替标点

换行不是逗号，不是句号，不是停顿。
每一次换行必须代表**真正的段落切换**，而非句子内部的语气停顿。

❌ **错误示范**（把一句话拆成碎行，信息量为零）：
```
他。

看了看。

那块石头。

很小。

但是。

很重要。
```

✅ **正确示范**（同等内容，正常书写）：
```
他看了看那块石头。虽然只有指甲盖大小，但他知道这东西的分量。
```

---

### 何时可以独行

只有以下两种情形允许极短行/单词独行：

1. **强节奏的戏剧转折点**：用极短行制造停顿感，每章不超过3处。
   例：`成功了。`（接在一段高强度动作描写之后）

2. **对话本身**：对话中每句台词独行是正常格式。

除此之外，任何"三个字以下独占一行"都是水字数信号，必须合并。

---

### 信息密度要求

每个自然段应包含**至少一个完整的事件/动作/情绪转变**，不得出现纯粹填充停顿感的碎行。

常见水字数模式，一律禁止：

| 水字数模式 | 正确写法 |
|-----------|---------|
| `但是。` 独占一行 | 并入上文或下文 |
| `他停下了。` + `然后。` + `继续走。` 各自独行 | `他停了停，继续走。` |
| 动作拆成3行：`他` / `抬起手` / `看了看` | `他抬手看了看。` |
| 情绪词单行：`愤怒。` / `绝望。` | 融入描写句中 |

## 检查清单

每次提交前自查：
- [ ] 章节名是本章爆点台词（不是总结标题）
- [ ] 无"第X章"等meta引用
- [ ] 无编造的不存在剧情
- [ ] 境界/战力设定合理
- [ ] 对话精简（≤3轮）
- [ ] 无武侠剧老套路
- [ ] 无换行当标点、无信息密度极低的碎行
